## 核心设定摘要

- 书名：归零
- 类型：科幻悬疑 + 都市日常 + 黑客文化
- 主角：林奕（中国留日大学生，CTF noob）
- 核心：虚拟网卡 wl0 = 时空 VPN 隧道。通过 CTF 渗透改变过去。
- 代价：蝴蝶效应——每次渗透波及周围的人。
- 闪回：déjà vu 碎片，不是完整记忆。
- 四卷：「连接」「提权」「越权」「归零」

## 创作原则

1. 故事和人物第一，读者第二
2. 蝴蝶效应有因果链，不是随机厄运
3. 大情感用小动作——用控制写失控
4. 量子术语只在高柳教授解释时出现
5. 唯一一抹奇幻：终端为何选中林奕——不解释

## 编辑工作流

```
确认写/改第N章 →
  ① 加载 harness SKILL.md（主编 Agent）
  ② 上下文 Agent 读取 状态/ + 记忆/ + 设定/大纲/
  ③ 写作/审稿 Agent 执行
  ④ 更新 状态/主角.md + 记忆/事件索引.md
```
