# Arch Linux

独立开发的 Linux 发行版。滚动更新（没有固定版本号，持续更新）。KISS 原则（Keep It Simple, Stupid）。特点：

- pacman 包管理器
- AUR（Arch User Repository）：社区维护的包仓库
- 安装是纯命令行——没有图形化安装向导
- i3/sway 等平铺窗口管理器在 Arch 用户中很流行

在《归零》中，林奕用 Arch。Ch1 开场就在配桌面（i3）。P53 前主人 waterlily 的系统也是 Arch。这不只是角色审美——Arch 用户天然和「自己编译、自己配置、自己负责」的黑客文化吻合。

调侃：「我用 Arch」——IT 圈最著名的 meme 之一。林奕从不说这句话。
