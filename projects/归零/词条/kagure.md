# kagure

Arch Linux 论坛上一个注册六年、发帖零、回帖零的账号。私信林奕删帖警告，之后开始在线教学。

教的不是理论——是渗透技术。用 CTF 题目教学，做完一题发下一题，只给点评不给讲解。风格冷、直接、不废话。

kagure 不是「老师」——是「一个比你早六年撞上同一堵墙的人」。

他的 ID 来自已故恋人神楽遥（Kagura Haruka）——把最后一个元音从 a 改成 e。不是拼错。是故意的。他用她的 ID 活在她来过的地方。

kagure 从未持有终端权限。他凭遥留下的笔记和日志碎片独自追查六年。他知道 wl0 的存在——因为他监测到了载波信号。但他不知道全部。他只知道一部分真相——比林奕多，比美佐少。
