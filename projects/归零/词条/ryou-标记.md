# Ryou 标记

千夏在服务器 hex tree 第五层目录中发现的一个注释。不是 hex 编码——是明文。写在最底层目录的注释里，只有一个词：

`Ryou`

林奕看到的时候没有立刻反应。然后他想起了 A 面刻字——「ハルカ と リョウ」。Haruka to Ryou。遥与凌。

这是神楽遥留下的标记。不是给林奕的——是给宫城凌的。遥在临终前把某些东西藏在了谦治的服务器里——她知道凌会追查到这里。

这个标记是服务器 hex tree 的线索收束点：它连接了 P53 的 A 面刻字（遥与凌）、kagure 的六年追查（凌追遥）、waterlily 的 ID（遥的用户名）、和千夏父母的秘密。
