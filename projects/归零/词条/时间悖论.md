# 时间悖论（Temporal Paradox）

时间旅行中因因果律破坏产生的逻辑矛盾。主要类型：

- 祖父悖论：你回到过去杀了祖父 → 你不会出生 → 你不能回到过去杀祖父 → 祖父活着 → 你出生了 → ...（循环矛盾）
- 本体悖论（Ontological Paradox）：信息/物体无起源（见 Bootstrap Paradox）
- 命定悖论（Predestination Paradox）：你回到过去阻止某件事——结果你的阻止行为恰恰导致了那件事发生

在《归零》中，两种悖论都被规避了设计：
- 祖父悖论：渗透不杀人不消失人——是改变因果链上的信息流。千夏的死亡可以绕开。
- Merge Conflict：救谦治 → 谦治不卖 P53 → 林奕拿不到 wl0 → 无法回去救谦治 → 死锁。这是 Bootstrap 的阻止机制——因果律不允许这个循环闭合。
