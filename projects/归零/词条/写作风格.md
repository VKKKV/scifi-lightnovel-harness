# 写作风格参考

## 双轨节奏

技术场景 → 龙族节奏（江南）
情感场景 → 三秋缒节奏
交界处 → 两者融合

---

## 龙族式（技术/渗透/动作场景）

核心：电影感。现代隐喻。通感。短长交替。

句式：
```
短-长-短。镜头推进。
「空气在他周围凝固了，像是整个世界按下了暂停键。然后他开口了。」
```

技法：
- 现代隐喻：「像是被随手扔在角落的易拉罐」
- 通感：「皮肤在尖叫」「声音在骨骼里震动」
- 对比结构：轻描淡写→压倒性效果
- 镜头法：内部→外部，环境→角色→感官细节
- 未完成的动作：「嘴巴张开了又合上了」

---

## 三秋缒式（情感/日常/孤独场景）

核心：极简。克制。留白。事实替代感受。

句式：
```
短。短。短。留白。
「他没有哭。他只是坐在那里。」
「她说什么都没说。但好像什么都说了。」
```

技法：
- 事实替代感受：「我的人生值三万日元」
- 对话无标签：直接接台词，不加「说道」
- 对称结构：「春天认识了她。春天失去了她。」
- 重复+变化：「什么都没说。什么都没说，但好像什么都说了。」
- 最重要的句子最短

---

## 禁止项

- ❌ 不写「他感到」「他觉得」「他的内心」
- ❌ 不写景物排比
- ❌ 不写大段心理独白
- ❌ 不写「说道」「喊道」「低声道」
- ❌ 不写段尾升华
- ❌ 不用「不到二十剑」式数字搭配
- ❌ 不写解释性旁白（「因为」「毕竟」）

## 归零应用示例

### 渗透场景（龙族式）
```
屏幕上的输出变成了一片噪声。
不是正常的网络噪声——是某种更底层的东西在碎裂。
像是有人把因果链拧成了麻花，然后用力一扯。
他能感觉到 P53 的键盘在震。不是错觉。是物理的。
```

### 情感场景（三秋缒式）
```
赵逸风把牛奶放在桌上。
凉的。
「你想说的时候。我在。」
林奕接过牛奶。喝了一口。
「谢了。」
```

### 交界处（融合）
```
git log 里的 commit 开始消失。
一条一条。像有人在逐行删除他的操作记录。
他看着屏幕。
没有敲键盘。
窗外的路灯还是橙色的。第一天晚上也是这个颜色。
然后屏幕黑了。
```
