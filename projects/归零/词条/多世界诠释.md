# 多世界诠释（Many-Worlds Interpretation, MWI）

量子力学的诠释之一。每一个量子事件中所有可能的结果都在某个分支宇宙中真实发生——宇宙不断分裂。不存在「坍缩」——所有可能性都实现了，只是我们只能感知其中一条。

在《归零》中，高柳教授的理论框架是退相干历史（Decoherent Histories），不是纯粹的多世界。区别在于：

- MWI：所有分支同时存在，观测只是让你看到其中一条
- 退相干历史：分支通过退相干坍缩——没被观测到的分支被「抹除」，留下闪回碎片

林奕的渗透操作相当于强制选择某条分支坍缩为现实。其他分支不是「平行宇宙」——是被剪掉的 git branch。
