# 退相干（Quantum Decoherence）

量子系统与环境相互作用导致量子叠加态丧失的过程。一个处于叠加态的量子系统——在未被观测时——会同时处于多个可能状态。一旦与环境发生不可控的相互作用（如被光子撞击、被仪器测量），叠加态就会「坍缩」到某一个确定的经典状态。

在《归零》的世界观中，高柳教授的框架将此概念扩展到宏观尺度：世界线分支不是平行宇宙，而是因果链的不同坍缩态。每次渗透操作触发退相干——从那个修改点往后的全部因果都重新计算。

通俗版：你有一个同时是 0 又是 1 的量子比特。你一碰它——它就只能是 0 或只能是 1 了。你改了一个时间点的数据——从那之后的历史就只能是那个版本了。
