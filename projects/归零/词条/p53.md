# P53

ThinkPad P53。林奕从浅羽商店仓库角落翻出来的二手笔记本。

配置：i9-9880H / RTX 5000 / 128GB / 12TB SSD / 4K Dolby Vision。电池槽空的。重量 3.2 公斤。

A 面（顶盖）右下角有银色细线体刻字：`ハルカ と リョウ`（遥与凌）。林奕能读出发音但不认识对应的汉字和人物。

原本是神楽遥在谦治的电脑店定制、要送给宫城凌的生日礼物。从未送达。遥死后，谦治按原地址寄给凌——无人签收，快递退回。在仓库落灰多年，被翔太当破烂卖了。

P53 不是终端——是终端的宿主。wl0 出现在这台机器上，但 P53 本身只是一台普通电脑。
