# 词条集

像文字冒险游戏中的关键词系统。每个词条是对小说中某个概念/物品/人物/知识点的简短解释。
写作时可作为参考，也可以在正文中以「角色查到的资料」形式自然嵌入。

---

## 使用方式

- 写作时需要确认某个设定的准确描述 → 查词条
- 需要给角色一段「查阅资料」的场景 → 直接引用词条内容
- 新概念首次出现时 → 创建新词条
- 读者视角的简明解释——不是设定文档，是「如果这本书有 glossary，这一条怎么写」
