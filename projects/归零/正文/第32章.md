# 第32章 · 深层

千夏把美佐的日志全部拷了出来——不是复制到自己的电脑上，是打印了。纸质。一叠 A4 纸，订书机订了两下，放在柜台下面的抽屉里。

林奕在 P53 上翻服务器深层文件的时候——不是千夏给的，是他用 wl0 隧道连到服务器本地网络，以 root 权限读取的文件系统——发现了一些不一样的东西。

不是 `.meta`。不是目录结构。是系统注释。

在 P53 上打开 hex tree 的某个深层文件时，终端输出了一段他没见过的东西——不是文件内容，是文件系统的扩展属性。

```
Subject: Asaba Misato
Role: Observer-07
Status: Terminated (cascade failure)
Residual: archived
```

林奕的手指停在键盘上。

继续往下翻。下一个文件。

```
Subject: Asaba Chinatsu
Role: Non-observer (legacy)
Status: Active
Flag: death_node (conditional)
```

千夏的名字。和她母亲的名字放在一起。

death_node。conditional。

有条件的。

他在 daemon.md 的规则里加了一笔。不是一行——是一个词，加在原来那条 death_node 记录的后面。括号。

`(conditional)`

他没有告诉千夏。不是想瞒。是他不知道条件是什么——告诉她也没有用。

***

下午。林奕在便利店。

千夏在看打印出来的日志。翻到一页停住了。

「她写到实验。」

林奕等她继续。

「2009 年的日志。她刚加入团队。她说——」千夏读了一行。「「今天第一次看到它运行。不是计算机。不是工具。是一个接口。通往一个我们不该打开的东西。」」

千夏合上打印纸。

「她知道自己不该打开。」

林奕没有接话。

千夏把打印纸放回抽屉。关上。

「但她还是打开了。」
