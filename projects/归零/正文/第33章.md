# 第33章 · 铅笔

千夏在整理储物间的时候翻到了一个纸箱。不是刻意在找——是冬天到了，她想把夏天的衣服收起来，冬季的被子拿出来。纸箱在壁橱最里面，压在两个行李箱下面。上面写着「本·大学·昔」。

谦治的笔记本。

旧纸箱的味道。翻开第一本——BBS 时代的技术笔记。服务器配置。网络拓扑。PHP 代码片段。字迹歪歪扭扭——谦治写字不好看，像小学生。千夏认得这个字迹。父亲在收银台后面记账的时候也是这种笔迹。

她本来想放回去。但翻到最后一本的时候，最后几页的笔迹变了。

更潦草。像是深夜写的。日期——谦治去世前几个月。

千夏蹲在储物间的地板上读完了。

她说不上来心里是什么感觉。不是震惊。不是悲伤。是一种很平的空——像你知道一件事很久了，只是没有证据。现在证据在手上了，你没有想象中的震动。你只是想：果然。

她在客厅坐了一会儿。然后把笔记本拍了照，发给林奕。没有附文字。

照片上谦治最后几页的内容：

`服务器上有些文件不是我创建的。目录名是乱码。`

`我没动它们。美佐的东西，我不碰。`

`有些 commit 不是我操作的。作者是一个我不认识的 ID。`

`我查了那个 ID。什么都查不到。像是从不存在。`

`美佐——你到底留了什么给我。`

最后一页。只有一行：

`千夏，别碰服务器。`

千夏蹲在储物间。地板上很凉。她已经碰了。而且她打开了父亲选择不碰的东西。

她把笔记本合上。放回纸箱。把纸箱推回壁橱最里面。

站起来。拍了拍裤子上没有的灰。

***

晚上。

林奕坐在公寓里。看着千夏发来的照片。谦治的铅笔字。他读了两遍。一个开电脑店的普通人——wol 的 BBS 站长——发现服务器上有不属于自己的东西。他追查了。什么都没查到。然后他写了一句话给女儿：别碰。

他不是在保护服务器。他是在保护千夏。

林奕把照片存在了一个单独的文件夹里。不是 daemon.md。是自己的笔记。

他不知道谦治在最后几个月里想过什么。不知道自己如果处在谦治的位置上会怎么做。但他知道一件事——谦治什么都不知道，但他做了他知道的最好的事。

把秘密带走。

林奕把手机放下。窗外路灯亮着。赵逸风在客厅喊了一句「这个 boss 打了三遍了过不去——」。日常的声音。

他把 P53 打开。不是为了渗透。是为了把谦治笔记本里的那个 ID——kiri——加入 git log 的搜索列表。

作者：kiri。和谦治笔记本里写的「我查了那个 ID」是同一个。

谦治查过 kiri。

什么都没找到。

林奕看着屏幕，然后敲了一行搜索命令。不是 git log——是搜索引擎。他搜了 kiri 这个 ID。

没有结果。

和谦治一样。
