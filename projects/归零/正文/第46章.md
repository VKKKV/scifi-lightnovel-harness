# 第46章 · 越权

林奕在便利店门口站了一会儿。

不是买东西。是看那辆白色轿车——今天不在。第四天。不在了。他不知道这意味着什么。是组织转移目标了，还是店主做了什么他不知道的事。

他推门进去。千夏在柜台后面。面前放着那本暗红色的笔记本——不是翻开的，是合着的。放在抽屉旁边。她刚看完。

林奕没有问。

千夏也没有说。

「今天有鲣鱼干饭团。」她说。像是每天说的那句话。

林奕从货架上拿了一个。放在柜台上。千夏扫了条码。

「门口的白色车没有了。」

「嗯。」千夏把饭团装进袋子里。「可能是放弃了。」

「也可能是换了一辆。」

千夏看了他一眼。没有反驳。

林奕走出便利店。十一月的风。天空很蓝。他打开 P53 看了看 git log。没有新 commit。没有 `挡`。没有 anomaly。

一切正常。太正常了。

他合上电脑。往公寓走。

街道上是普通的早晨。送报纸的摩托车。便利店开了。自动贩卖机在发光。没有人知道一群人在这个城市的几个角落里，被一个他们无法理解的东西追着。

他走着。手里拎着那个鲣鱼干饭团。

P53 在包里。A 面刻着两个名字。

ハルカ と リョウ。

他加快了一点脚步。不是因为有人在追他——是想早点回去。

赵逸风大概已经泡好面了。

卷三·越权 · 完
