# 第8章 · grep

千夏学了三周，已经能在终端里做三件事了。

`ls`——看目录。`cd`——换目录。`cat`——看文件内容。加起来不到十个字母。但她用这三个命令做到了一件林奕没预料到的事。

她开始记录服务器的文件结构。

不是用 `tree` 命令——她不知道有 `tree`。是手工的：`cd` 进一个目录，`ls` 看内容，把名字和日期抄在本子上。一层一层。到第三周的时候，她的笔记本画满了一棵手绘的目录树——从根目录开始往下，每个分支旁边标注了文件数量和最后修改日期。有些目录她打不开——权限不够。她在旁边画了问号。

「这就是你知道的全部。」林奕看着她的笔记本。

「不全是。」千夏翻到后面一页。她画了一个表格——日期、指示灯颜色、硬盘声音。从林奕来东京的第一个月开始。每天记。没有间断。「这些我看得懂。灯的颜色和硬盘读写——有时候半夜会变化。」

「变化规律是什么。」

「不知道。」她在表格最下面一行点了点——那是两周前的一个日期，标注着「灯全绿——持续三小时——硬盘读写很频繁」。日期是林奕执行第一次渗透的第二天。

林奕看着那行。什么都没说。

千夏也不知道这意味着什么。她只是记。

***

同一天下午，林奕在杂货店柜台后面吃晚饭。

翔太今天没排练——乐队主唱感冒了。他在柜台后面调弦，还是那把木吉他，还是副歌前停。林奕坐在柜台外面——一个倒扣的啤酒箱上。手里是一碗从隔壁中餐馆买的炒饭。

「你那个电脑——」翔太低头调弦，没看林奕，「是不是有什么东西。」

林奕的筷子停了一下。

「不是坏的——」翔太弹了一个和弦。大三和弦。听起来很亮。「是那种……我说不清。有时候你不在，千夏上楼看服务器。然后我在这边看柜台。电脑在二楼——但我能听到。风扇有时候突然转很快。不像正常的快——是那种『你在跑什么』的快。」

林奕嚼完嘴里的炒饭。「你懂电脑？」

「不懂。」翔太换了一个和弦。小七。比刚才暗。「但我懂乐器。乐器和电脑一样——正常运转和不正常运转的声音不一样。你那台——千夏爸爸的那台——最近不正常。」

「怎么不正常。」

翔太想了一下。「不是坏了。是有人在用。」

他看着林奕。眼影今天没补——早上画的已经掉得差不多了，只剩眼角还留着一层很淡的灰色。他的眼睛没有笑——不是平时那种「我亏了但我不是很在乎」的轻松。是认真的。

「我不知道你在做什么。但别让她出事。」

林奕把筷子插在炒饭里。

「不会。」

翔太点了一下头。继续调弦。

***

晚上林奕在公寓里翻 git log。

自动 commit 的密度增加了。之前每天几条——现在每小时都有。不只是他和千夏的互动——杂货店顾客进出、卡车路过、街上某个人的手机在路口连了一下 Wi-Fi。系统在记录一切。

但有一条不一样。

不是 `observed`。commit message 解码之后是另一个词：`anomaly`。

异常。

林奕展开这条 commit。内容不是网络流量——是他自己的终端输出。凌晨三点零四分。他在睡。但 P53 的终端上出现了一行输出——系统自己产生的。

`wl0: link up.`

就这一行。在凌晨三点。没人碰键盘。

wl0 自己亮了。

林奕盯着屏幕。wl0 从第一天起就是 UNKNOWN 状态。不能 `up`。不能连接。不能通信——除了他自己主动发起的那几次渗透。但凌晨三点——系统自己把它激活了。

他翻到下一行。凌晨三点零四分零七秒。又一 commit。

`wl0: carrier signal detected. source: unknown.`

载波信号。检测到。来源未知。

然后凌晨三点零四分十二秒——第三条 commit。

`wl0: packet received. dropped — not addressed to current holder.`

收到一个包。丢弃——不是发给当前持有者的。

林奕靠在椅背上。折叠椅响了一声。赵逸风已经睡了。

不是发给他的。是发给别人的——但 wl0 的持有者只有他一个。

他把这几条 commit 截图存进 daemon.md。在下面加了一行。

`规则 8：wl0 不是只有我能用。有人在别的终端上——或者在别的时间上——也在用它。凌晨三点零四分十二秒，一个包经过了 wl0。不是给我的。`

窗外没有路灯了——已经过了路灯关闭的时间。整个房间只有屏幕的光。林奕的脸被映得很白。

他忽然很想给赵逸风留一碗泡面。

但他没有。赵逸风在睡觉。他什么都不知道。

林奕关掉 daemon.md。打开一个新的文档。标题打了两个字：千夏。

然后他删掉了。没写。

不知道写什么。她不知道他救过她。不知道那辆卡车为什么撞她。不知道她父亲留下的服务器在凌晨三点会自己跑起来。不知道林奕的电脑上有一个包是发给别人的——不是给他的。

她只是在记指示灯的颜色。

林奕合上屏幕。睡觉。

***

第二天下午，他在杂货店外面写代码。

不是渗透用的——是一个 Python 脚本，帮千夏自动扫描服务器的文件结构。用 `os.walk()`，输出一个树形图。代码很短，不到五十行。但他在 `filter` 那行卡住了——语法写错了。查了三遍文档都没发现。

一个声音从他后面传来。

「你这个不对。」

女生。不是千夏。千夏的声音他认得。

林奕回头。

一个穿黑色连帽衫的女生站在他身后。帽子没戴——头发很短，刚到耳朵。背着一个帆布包，包上别着几个 CTF 比赛的 pin——HITCON、SECCON、DEF CON CTF。手里拿着一罐咖啡。没在看他——在看他的屏幕。

「不是排除隐藏。你用的是 Python 不是 shell。点号在 Python 里不会自动表示隐藏——你要用 `os.path.basename`。或者 `startswith('.')`。」

她说完就走了。

没停。没回头。

林奕看着她的背影——帆布包上的 pin 在杂货店荧光灯下反光。她推开杂货店的门。门铃响了一声。进去拿了一瓶茶。出来。

从林奕身边走过去。又扫了一眼他的屏幕。

「filter 写错了。」她说。然后走了。

这次林奕注意到了——她的眼睛扫屏幕的速度很快，不是读代码——是扫，和他在 OSINT 里扫街景找位置一样的方式。不看全部。只看重要的。

他还没来得及说谢谢。她已经走到街角了。黑色连帽衫融进人群里。

林奕低头看自己的代码。`filter` 那行是错的。她说的对。

他在脚本最上面加了一行注释。

`# 修 filter。连帽衫女生说的。`
