# 第51章 · 条件

林奕的渗透操作暴露了终端的位置。

不是 IP 地址——是物理位置。组织检测到 wl0 的活动频率和模式——不是某一次，是长期积累的轨迹。他们不知道终端在哪栋楼。但他们知道在哪个区。

林奕在 git log 里看到了那条 commit。不是他操作的。不是系统自动的。是组织——通过某种方式——在 git log 里留下了一条记录：

`death_node: Asaba Chinatsu. Status: conditional → active.`

他看到的时候已经是 Commit 产生后的第几个小时。条件揭晓了。death_node 的条件——不是世界的随机性。是组织检测到终端活动 → 定位 → 出手。千夏因为和终端的关联成为目标。

他拿起手机。打给千夏。不是发消息——是做的那件事。

响了三声。接了。

「喂。」

「你在店里吗。」

「在。」

「别出来。」

千夏沉默了一秒。「为什么。」

「门口是不是有一辆白色的车。」

千夏没有回答。但林奕听到了话筒那边——她拉开了柜台的抽屉。又关上。

「你要我做什么。」

「关好门。等我到。」

林奕挂断电话。拿上 P53。外套没有拉——已经在门口了。

***

他到便利店的时候白色的车还在。停在路对面。车窗是黑的。他走进去。千夏在柜台后面。X230 开着。她没在写代码——在看门口的监控画面。

「它没动过。」

「停了多久。」

「你打电话之前十分钟。」

林奕站在柜台前。P53 在包里。拉链没拉好。他不知道组织的人在哪辆车里——或者不在车里。他们可能在任何地方。那张窗户后面。那栋楼的楼顶。那台自动贩卖机旁边低着头看手机的人。

他从来不知道组织长什么样。

但组织知道他在哪了。

他在便利店待到打烊。千夏没有问他要不要回去。她多待了一小时。

关门的时候林奕走了出去。白色的车还在。

他往公寓走。没有回头。

凌晨。他坐在 P53 前。daemon-v2.md 里多了一行：

`death_node 的条件：组织检测到终端活动。千夏因为我而被标记。`

他敲完，停了一下。

下面还有一行。他没有删——但他写下去的时候手停了半拍：

`我救她——她因为被救而被标记。我不救她——她死在卡车下。两条路都是死路。`

他关了电脑。

窗外的路灯亮着。和第一天晚上一样。但一切都不一样了。
