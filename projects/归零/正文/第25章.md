# 第25章 · 泡面

赵逸风收到了一封邮件。

不是他的大学邮箱。不是学校通知。是一个他不认识的地址——一串数字和字母的随机组合。主题栏只有一个词：

`wl0`

他本来不会注意到。垃圾邮件每天都有。但他看到了那三个字符——他在林奕的屏幕上见过。那个黑窗口里反复出现的名字。wl0。

他点开了。

邮件正文是一段加密文本。没有上下文。没有说明。没有署名。只有一段他完全看不懂的东西——base64 或者类似的东西，他分不清。

赵逸风盯着那封邮件看了一会儿。

他没有问林奕。他把邮件转发给了林奕。没有加任何说明。转发。关掉邮箱。继续吃薯片。

***

林奕看到那封转发邮件的时候是凌晨三点。

他刚从便利店回来。P53 上 git log 的自动 commit 还在刷。他打开手机——看到赵逸风转发来的邮件。发件人的地址里有 wl0。正文是一段他认识的编码格式——和 kiri 的 commit message 是同一类。

他坐在折叠桌前。手里拿着手机。

赵逸风在客厅。泡面的味道飘过来了。没说话。两个人隔着一道墙。谁都没有敲对方的门。

林奕不知道自己坐了多久。

然后赵逸风的脚步声从客厅响起来——走到他桌前。一碗泡面放下来了。不是推——是放。筷子搁在碗边上。对齐碗沿。

赵逸风没有看他。没有说话。放完泡面转身回房间了。

林奕看着那碗泡面。

蒸汽从杯面的小孔里冒出来。海鲜味。玉米粒浮在汤上面。他认识这个泡面——赵逸风抢的联名款。他自己一直没舍得吃。

现在放在他桌上。

他不知道赵逸风懂多少。不知道他有没有搜索 wl0。不知道他有没有试图解密那段文本。不知道他为什么选择转发而不是问他。

但他知道一件事——赵逸风懂的不是计算机。是人。他知道林奕在承受什么。即使他不知道是什么。

林奕把泡面吃了。

汤喝完了。筷子放回碗里。他没有去回复那封邮件。没有去查发件人。他做了一件他也没想到的事——走到赵逸风房间门口。站了一秒。

然后回了自己那边。

没有敲门。

没有说谢谢。

凌晨三点半。两个房间的灯都亮着。谁都没有关。
