# 第13章 · wall

下周三之前隔了五天。五天里林奕哪都没去——除了便利店和语校，他坐在 P53 前。不是渗透。是在翻 git log。

他把高円寺那晚之后的所有 commit 看了一遍。高柳的名字没有出现。高柳的 commit 不会出现在这里——高柳没有终端。但林奕想确认一件事：凌晨三点零四分的那个包——目的地 waterlily——在他说出「六年」的时候有没有变化。

没有。waterlily 还在。凌晨三点零四分还在。什么都没变。

他把 P53 合上。窗外太阳快落了。赵逸风的直播间声音从客厅传过来——今天是粉色头发那个，在玩开放世界游戏，迷路了，同一个山洞转了三圈。

「你说——」林奕没转头。

「嗯?」

「有一个人研究了六年。他研究的东西——他不知道是什么。但他研究它。」

赵逸风暂停了游戏。等了一会儿。「然后呢。」

「没有然后。就是——六年。」

「嗯。」赵逸风想了想。「那个人还在研究吗。」

「在。」

「那就不是六年。是还没结束。」

林奕没接话。他看着 P53 A面上的刻字。Ryou。四年了。Ryou 的终端——如果这个名字没猜错——比他先拿到这台 P53。然后呢。Ryou 去了哪里。为什么留给他。

他把刻字那行拼音抄在便签上。贴在屏幕旁边。和 kagure 的私信贴在一起。

***

周三。B1 的营业时间是晚上七点。林奕五点就到了。

门锁着。没有灯。招牌没有亮。他站在下北泽的窄巷子里——昨晚的雨还没干透，地面是深色的，映着路灯刚亮的光。

六点零三分。高柳从巷子口走过来。不是从车站方向——是从反方向，手里拎着一个便利店的袋子。他看到林奕，没有惊讶。

「早了。」

「嗯。」

高柳从口袋里拿出钥匙。不是普通钥匙——是那种老式的长柄钥匙，像是 B1 刚开业的时候用的。他把锁打开。推门的时候金属门框蹭了一下地，发出一声短而哑的响。

「进来。」

B1 没有开灯。高柳走到吧台后面，按了电闸开关。天花板上的荧光管一根一根地亮起来——不是全亮，大概一半。光不够均匀，照在空荡荡的舞台和散落的椅子上。

没有演出的时候，B1 像一个废弃的防空洞。

高柳把便利店袋子放在桌上。里面是两罐咖啡和一袋面包。他把一罐咖啡推到林奕面前。

「坐。」

林奕坐下。高柳没有坐吧台后面——他拉了对面一把椅子，坐在林奕前面。没有桌子隔着。两个人之间只有空气和荧光管的光。

「你帖子上描述的那个网卡现象——」高柳打开自己的咖啡罐。喝了一口。「我研究了六年。」

六年前。2019 年。和 wl0 MAC 地址末尾的那个日期一样。

「你也有终端?」

「没有。」高柳把咖啡罐放下。「我没有见过那个接口。但我推导过它存在的依据。」

他的研究方向是量子信息科学。不是终端——是退相干历史。世界线分支理论。他把手机拿出来，翻到一篇论文，把屏幕转过来给林奕看。物理学综述，公式占了大半页。林奕看懂了标题——关于退相干历史中信息残留的数学模型。

B1 的荧光管闪了一下。远处的巷子里有电车经过的声音。高柳的咖啡罐在桌上转了一圈。停在林奕那罐旁边。

「你描述的现象——如果真实存在——那是退相干的宏观可观测效应。」高柳收回手机。「超光速不是必要条件。分支之间的信息泄漏，在数学上有一个被大多数人忽略的解。」

「多数人不认为那个解真实存在。」

「对。」高柳喝了一口咖啡。「但我研究了六年，因为有人用事实让我相信它存在。」

「你看到的那些——不是超自然。」高柳的手指在桌上点了一下。「是物理。是我们还没建好观测工具的物理。」

***

高柳没有问终端是什么、怎么运作。他问的是数据。

「git log 的格式。commit 的频率。anomaly 标记的模式。」

林奕把 P53 打开了。他把 daemon.md 翻出来。十七行规则。最新的几条还带着问号。

高柳没有拿电脑。他让 P53 放在桌上，屏幕朝着他。读完了。没有评价。

「你从来没有想过——」

「什么。」

「为什么你的 git 命令在世界线上有用。」

林奕愣住了。他确实没想过。

高柳没有追问。

***

然后高柳说了一句话。语气平淡。像是在说一个不重要的技术细节。

「你有没有试过 `git branch`。」

林奕没有。

他从来没想过。git log 一直有输出——他一直在看 commit。但 branch？他不敢。他不知道 git branch 在这个语境下意味着什么。是分支世界线吗。是他还没见过的分支。是他不想见的分支。

「别试。」高柳说。语气仍然平淡。「先搞清楚 `git log` 能告诉你什么。分支——是后面的事。」

「你不知道我能看到什么。」

「我知道。」高柳喝完最后一口咖啡。把空罐放在桌上。「因为你如果看到了——你现在不会坐在这里。」

***

离开前高柳说了最后一句话。

「你的 kagure——如果他是我说的那个人——他不会告诉你全部。因为他自己也不知道全部。他只是比你早六年撞上这堵墙。」

「你不认识他。」

「不认识。」高柳站起来。把空罐扔进垃圾桶。「但你的 MAC 地址末尾是一个日期。2019 年 12 月 24 日。我研究开始的那一年。」

「你查过那条 commit。」

「我研究了六年。」高柳往门口走。推开门之前回头看了一眼。

「他不是不告诉你。是他能告诉你的——都已经告诉你了。剩下的他自己也不知道。」

***

林奕走出 B1 的时候天已经完全黑了。

下北泽的巷子里路灯亮着。居酒屋的暖帘已经挂出来了。几个人站在巷口吸烟。普通的周三夜晚。

他把 P53 的充电器往肩上拢了拢。往车站走。

脑子里不是高柳说的那些理论。不是 git branch。

是高柳说的最后那句话。不是对他说的——是对自己说的：

「他不是不告诉你。是他能告诉你的——都已经告诉你了。」

林奕不知道 kagure 是谁。不知道他长什么样。不知道他在哪个城市。但他知道六年前有人做过同样的事——敲了一行不认识的命令，然后世界变了。

那个人用剩下的六年做了两件事：追查真相。警告后来者。

没有第三件。

林奕走到车站。掏出手机。论坛私信界面还开着。kagure 的头像是灰色的。最后一次上线时间——三天前。

他打了一行字：

`你见过高柳吗。`

发送。

他把手机放回口袋。电车进站了。车门打开的时候风从隧道里灌出来。

他没有等到回复。
