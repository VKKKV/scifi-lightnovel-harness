# 第38章 · 提权

从 B1 回到公寓之后林奕睡了一整个白天。

不是累——是用尽了。从凌晨四点看到 anomaly 到站在 B1 黑暗里听店主说那句话——二十个小时。他倒在床上。外套没脱。醒来的时候窗外是黑的——不是晚上，是傍晚。赵逸风在客厅，声音压得很低，在打电话。

「……嗯。醒了。……没事。他昨天出去了。……不知道。没问。」

林奕躺在床上。听着赵逸风压低的声音和窗外十一月的风声。

他把 P53 打开。没有新 commit。没有 anomaly。一切正常。

他打开 daemon.md——新 daemon.md。从零开始的那本。现在已经有十几条了。他翻到最后。空白的。光标在闪。

他打了一行字：

`凌说 kiri 的 commit 出现在坍缩之前。不是记录——是预测。`

然后他停了一下。又打了一行：

`B1 店主说 P53 不是第一次被带进那里。有人在我之前到过 B1。可能是 Ryou。可能是 kiri。可能是同一个人。`

他保存了文件。

然后他做了一件之前没做过的事——在文件名后面加了一个后缀。不再是 `daemon.md`。他把它改成了：

`daemon-v2.md`

不是因为版本号。是因为第一本 daemon.md 他删了。那些规则还在他脑子里。但规则不再是全部了。他不知道的比知道的更多。

他把 P53 合上。

凌说的最后一句话还在脑子里——“你什么都不知道。才刚刚开始。”

他拉开窗帘。窗外的东京。十一月的傍晚。路灯刚亮。赵逸风挂断了电话。「吃面？」

「吃。」

客厅里传来撕包装袋的声音。热水壶烧开了。蒸汽从壶口冒出来。他坐在林奕对面——不是问昨天去了哪里。

「今天咖喱的。」

「嗯。」

泡面的味道和傍晚的风混在一起。他坐在桌前。P53 在桌上。A 面朝上。ハルカ と リョウ。刻字在灯光下几乎看不到——只有特定角度才反光。

他看了一会儿。然后低头吃面。
