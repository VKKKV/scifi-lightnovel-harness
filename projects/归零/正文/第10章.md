# 第10章 · whoami

林奕的技术瓶颈出现在第十一次渗透之后。

他想黑进一个医院的内部系统——不是做坏事。是他在 git log 里看到了一条 commit：三个月前有人被误诊，正确的诊断结果被压在系统里没发出去。他想把那条通知补发。但医院的内网有 WAF，有内网隔离，有 IDS。他改摄像头时间戳进路口网络那一套完全不适用。

他在自己的笔记里翻了三遍。没有解决方案。搜了谷歌。没有——或者说，搜出来的东西太多了。SQL 注入、XSS、CSRF、内网穿透、横向移动。每个概念他都认识。但不知道怎么串起来。就像一个认识所有零件但装不出引擎的人。

他需要有人告诉他顺序。

但林奕不爱问问题。他这辈子在论坛上发过的帖子不超过五篇——全是 OSINT 图寻挑战的答案。求助帖——零。

这次他发了。

在 Arch Linux 论坛的网络安全板块。标题：「未知虚拟网卡——驱动不存在，但 `ip link` 可见」。正文描述了 wl0 的现象——MAC 地址格式不对、ethtool 不识别、没有 PCI 设备、没有内核模块。他没提渗透。没提时间隧道。只说了网卡。

帖子发出去不到三分钟。被删了。

林奕刷新页面。帖子不存在。回收站里也没有——不是移到回收站，是完全消失。就好像它从来没存在过。

论坛右上角的私信图标多了一个红色的 1。

他点开。

`删了你的帖。别再发。别再碰那个接口。`

发送者的 ID：kagure。

没有头像。注册时间六年前。发帖数：零。回帖数：零。除了这条私信，这个账号在论坛上没有任何痕迹。但它存在了六年。

林奕盯着那个 ID。

kagure。不是英文。他搜了一下——日语。神楽→kagura，但他改了一个字母。

他回复。

`你是谁。`

没有回复。

一天。两天。三天。林奕每天打开论坛看私信——那个红色 1 早就消失了。kagure 的头像始终是灰色的。不在线。

第四天晚上。一点。林奕正在翻 git log——他最近养成了睡前翻一遍的习惯，检查有没有新的 `anomaly` 标记。论坛页面在另一个标签页。他随手刷新了一下。

私信图标红了。

`你在用 wl0。`

林奕的手指在键盘上停了一秒。

他没在任何地方提过 wl0。帖子里描述的是网卡现象——ip link、MAC 地址、lspci。没有用过这个词。

他回复。

`你怎么知道 wl0 的名字。`

这次回复来得很快。不到一分钟。

`因为你发帖那天——我监测到了它的载波信号。你知道那意味着什么吗。`

林奕不知道。但他知道凌晨三点零四分那个包。

`你就是那个发包的人。`

`不是发。是收到了。你用那个接口做了什么——我这边全看得到。`

凌晨三点零四分。那个包。不是发给他的。是发给——不，是被监测到的。

林奕打字。删掉。又打。又删掉。最后只发了一句。

`你是谁。`

这一次 kagure 的回复等了很久。林奕以为他下线了。过了将近五分钟，回复来了。

`一个追了六年的人。`

然后头像灰了。无论林奕再怎么发消息，都没有再回。

***

第二天晚上，林奕坐在 P53 前。论坛的私信页面还开着。kagure 灰色头像旁边显示着最后一条消息的时间——凌晨一点四十三分。

「一个追了六年的人。」

林奕把这句话写在便签上。贴在 P53 屏幕旁边。

六年。这台 P53 是三年前组装的——2019 年末。如果 kagure 追了六年，那他追的不是这台电脑。是电脑背后的东西。wl0。wl0。

林奕把论坛私信的内容存进 daemon.md。在下面加了一条。

`规则 10：有人知道 wl0。不只是知道——在监测。我能被探测到。`

***

第三天晚上。私信又响了。

林奕几乎是立刻切到论坛页面。kagure 在线。这次发的不是警告。

`你现在能做什么。`

林奕犹豫了一下。然后敲了实话。全部。渗透。蝴蝶效应。卡车。千夏。删帖测试。凌晨三点 signal。全说了。他太需要有人告诉他顺序了。

kagure 过了很久才回复。只有一句话。

`你做的全是错的。继续下去，你救的那个人会死。每一次渗透都在积累。因果律。不是系统在监视你——是你自己。`

林奕看着屏幕。

`怎么不积累。`

`学。`

然后 kagure 贴了一个链接。不是教程。是一个 CTF 题目——一道内网渗透题。靶机地址。没有提示。没有步骤。

`做完这题。然后我再告诉你下一步。`

林奕盯着那个链接。点开。注册账号。启动靶机。

第一题花了他六个小时。不是难——是他不认识那些工具的先后顺序。nmap 扫端口。发现 80 开着。Web 页面有一个上传功能。传了一个 PHP shell。拿到了 Web 权限。然后要提权——但内核版本太新，脏牛用不了。他卡了两个小时。凌晨两点。他终于找到了一个 misconfigured sudo 配置——`www-data` 可以 `sudo tcpdump`。他用 tcpdump 抓了半天——全是常规流量。几乎要放弃了。然后他在抓包里看到一行异常的进程启动参数——管理员在半夜登录时把密码敲进了命令行。Root 登进去了。

Flag：`kagure{welcome_to_the_real_ctf}`

他把 flag 贴在私信里。kagure 回了。

`及格。下一题。`

没有夸奖。没有「做得不错」。就是「及格」和另一个链接。

林奕看着那个新链接。笑了——不是咧开嘴的笑。是那种你终于找到了一个人能说「你这个 filter 写错了」的笑。不需要对方对你好。需要对方对。

他打开第二题。凌晨两点四十分。

窗外没有路灯。只有 P53 的屏幕在闪。赵逸风的房间里传来翻身的声音——弹簧床垫的响声。然后安静。

林奕敲了第一行命令。
