# 第48章 · 自己

林奕在 git log 里追踪那辆卡车。

不是第一次查——他查过很多次。每次都被时间偏移挡住——commit 的时间戳指向未来，他看不懂。但现在他知道了怎么读美佐的笔记。系统的时间戳不是 UTC——是因果链的相对偏移。正数 = 过去，负数 = 未来。

那辆卡车的 commit 是负数。

他继续追。找到了 IP 来源——不是某个服务器。不是某个黑客。是 P53 自己的接口。wl0。来源地址：`wl0::outbound::t+???`。时间偏移指向未来的某一个点。他追不到那个点的具体内容——因为那个点还没有发生。

但他追到了另一条链。

奶龙贝利亚鬼畜图片。第一天。那张弹出来的图。吓到赵逸风打翻可乐的图。

来源也是同一个接口。wl0。也是未来的操作。

他坐在 P53 前。屏幕的光照在他脸上。

卡车是他。鬼图也是他。

不是有人要害千夏。不是有人在追踪他。是他自己——在未来的某一次操作中——产生了连锁反应，传回过去，让卡车闯了红灯。同样的，他在未来的某一次操作中，发了一张图给过去的赵逸风，吓他打翻可乐，让他去买 P53。

他自己启动了自己的故事。他自己杀了他要救的那个人。然后他回到过去救了她——因为他杀了她。

他在 daemon-v2.md 里打了最后一行。不是规则。不是观察。是一句话：

`卡车是我。鬼图也是我。`

他把 P53 合上。窗外天快亮了。赵逸风的闹钟还有一个小时才响。他听着赵逸风的鼾声——这个鼾声是他自己用一个还没发生的操作、一张还没发出的鬼图、一个尚未做出的选择创造出来的。

赵逸风的存在——他自己的存在——千夏的两次死亡——整个故事——是一团自我缠绕的因果链。他在每一段都既是因也是果。

他不知道该怎么办。

但他知道自己坐在 P53 前。天快亮了。赵逸风还在睡。

那行命令——和第一天一样——还在下一行等着他。
