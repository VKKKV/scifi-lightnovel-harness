# 第36章 · 六年

秋叶原的电子一条街。凌晨五点。天没亮。路灯还亮着。街边有几家通宵营业的店铺——一家卖二手相机的、一家卖模型的、一家在地下室的网咖。

林奕走下楼梯。

网咖的前台没人。空气里有旧烟味和空调的干燥气味。走廊尽头的小隔间里，有一台屏幕的光透过半透明的玻璃门漏出来。

他推开门。

角落里坐着一个男人。二十八九岁。黑眼圈重得像是没睡过觉。桌上有三台笔记本叠在一起——两台 ThinkPad，一台 MacBook。屏幕全是终端窗口。嘴里叼着一根没点的烟。

「你是 kagure。」

「你是那个用 P53 的蠢蛋。」

声音是哑的。不是感冒——是长期不说话的那种哑。凌看着他。没有站起来。没有伸手。只是坐在椅子里，像是坐了六年终于等到一个人推门进来。

林奕把 P53 从包里拿出来。放在桌上。

厚重的黑色机身。ThinkPad 的经典设计。放在三台轻薄笔记本旁边像一块砖头。

凌的目光扫过那台电脑。然后停住了。

A 面右下角。灯光昏暗。但银色刻字反了一瞬间的光——ハルカ と リョウ。

凌的手停了。不是抖——是停。叼着的烟从嘴角滑下来。掉在桌面上。烟灰散开。他没有去捡。

他盯着那行刻字。

沉默了很久。

「这台电脑——」他的声音变了。不是之前那个冷的、疲惫的声音。是更轻的。像是怕说大声了会碎。「在哪找到的。」

「浅羽商店。仓库角落。」

凌闭上眼睛。睁开。又看了一遍刻字。

「她给我做的。」他说。不是对林奕说。是对那行刻字说。「我从来不知道。」

他把掉在桌上的烟捡起来。没有点。捏在手指间。捏得很紧。

「她叫神楽遥。我的恋人。」凌的声音恢复了。但不是之前的疲惫——是另一种。平。太平衡了。「六年前。她死了。我没有去过那个仓库。没有见过这台电脑。她没有告诉我她做了什么。我追了六年——追的是她留下的东西。」

他把烟按灭在烟灰缸里。按得很慢。

「然后你告诉我——她六年前就在我面前。在浅羽商店的仓库角落里。我不知道。」

林奕没有说话。没有什么可说的。

凌看着他。看了一会儿。然后把烟灰缸推到一边。打开其中一台笔记本。屏幕上是一个终端窗口——git log。

「你那个 kiri 的理论——记录坍缩的系统自记录——」凌没有抬头。「我六年前也想到了。然后我花了三年证明自己错了。」

「为什么。」

「因为 kiri 的 commit 不是坍缩后才出现的。是坍缩前。」

林奕愣住了。

「它在坍缩发生之前就记录了。像是——它知道坍缩会发生。」

凌抬起头。屏幕上那行他追了六年的 commit 作者栏闪着光。

kiri。

林奕站在凌晨五点的秋叶原地下网咖里。三台笔记本。一个追了六年的人。一台他随身背着的、A 面刻着两个名字的 ThinkPad。

他突然觉得——他什么都不知道。才刚刚开始。
