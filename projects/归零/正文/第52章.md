# 第52章 · 回溯

那一天不是特别的一天。

没有组织突袭。没有新的死亡标记。没有凌的突然出现。只是普通的一天——林奕坐在 P53 前，千夏在便利店，赵逸风在客厅，叶岚在实验室，高柳在 B1。

林奕在做一次渗透。微小的操作——改一封从未送达的邮件的时间戳。不是重要的邮件。他只是想让某个人早一天收到通知。连锁反应最小。安全。

但在他敲下回车的那一刻——世界开始碎了。

不是物理上的碎裂。是因果链在坍缩。git log 里的 commit 开始消失——不是一条一条地消失。是一整片一整片地消失。屏幕上的终端输出变成噪声——ASCII 字符散落了一屏然后静止了。

林奕看着屏幕。手指没有离开键盘。

然后是安静。

不是环境安静——是一切安静了。窗外的风停了。赵逸风的直播停了。公寓楼上住户的脚步声停了。世界像是被人按了暂停。

然后一条 commit 出现在空白的终端上——不是 git log 格式。是一行纯文本：

`CASCADE THRESHOLD REACHED. INITIATING WORLD LINE COLLAPSE.`

林奕没有动。他看着那行字。

然后终端黑了。

不是关机。是输出停止了。光标在屏幕左上角闪了一下。静止了。然后——

林奕听见了键盘声。

不是自己的键盘——是另一种。旧的。机械轴。服务器机房里的声音。然后他看见了一个画面——白色的荧光灯。货架少一排。千夏不在柜台后面。闪回。但不是碎片——是完整的。他知道这个瞬间。

然后一切黑了。

***

林奕醒来的时候不知道自己在哪。

天花板不认识。窗外的光线不认识。空气的味道不认识。他躺在床上——一张他没见过但身体记得的床。桌上没有 P53。没有笔记本。只有一台旧手机和一个钱包。

他坐起来。

房间是单身公寓。比之前小。窗外是东京的普通街道。十一月的阳光。他看了一下手机。日期。

时间回到了一切开始之前。

他不认识千夏。不认识凌。不认识叶岚。不认识高柳。P53 在浅羽商店的仓库角落里落灰。一切都还没有发生。

但林奕记得一切。

旧世界线的全部——千夏的脸、凌的手停在刻字上、赵逸风凌晨三点的泡面、高柳在 B1 地下室的沉默、店主擦杯子的动作。全在他脑子里。完整的。没有碎片——是完整的记忆。

他坐在床边。手机在手上。时间还早。他可以做任何事。

他记得一切。但没人认识他。

他不知道自己坐了多久。然后站起来。洗脸。穿外套。出门。

不是去浅羽商店。不是去找 P53。是往语校的方向走。脚步和之前的每一天一样。但脑子里——他和某个人在某条已经不存在的世界线上一起做过的事像是他自己的回声。

他走着。十一月的东京。天空很蓝。一切和之前一样。

什么都没有发生过。但什么都留在了他脑子里。
