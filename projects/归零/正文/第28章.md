# 第28章 · 消化

B1 摊牌后的第二天。

没有人提起。便利店照常开门。语校照常上课。赵逸风照常在客厅看直播。一切和前一天一样。唯一的变化是——每个人都知道了一些不该知道的事。

千夏在柜台后面看的是密码学教材。不是林奕推荐的——是叶岚发来的链接。`Crypto 101`。PDF 格式。她在 X230 上一页一页地翻。没有问林奕这个函数是什么意思、那个算法能不能用。她在自己看。

林奕坐在她对面。P53 开着——但他在看千夏。不是看她这个人——是看她翻 PDF 的速度。她不是慢读型。她扫一眼，停一下，想一会儿，然后往下翻。像在确认自己理解了才继续。

「这个网站。」千夏没抬头。「注册要邮箱。你有建议吗。」

「用 Proton。」

「好。」

她注册了。开始在 CTF 平台上做密码学题。第一道——Base64 解码。她花了三秒。第二道——ROT13。花了五秒。第三道——Vigenère。她停了两分钟。然后解出来了。

她在本子上记了一行：`Vigenère 的密钥长度用 Kasiski 检验。`

本子上已经有了之前的树形图。现在旁边又多了一行密码学笔记。

***

叶岚发了邮件。林奕看到的时候在便利店。附件是一张图——横轴是时间，纵轴是分支密度。她在邮件里写了一段话：git log commit 密度有周期性，像潮汐。不是因为林奕在操作，是系统自己在某个周期律里涨落。

林奕看了那张图。他不知道这意味着什么。他把邮件转发给了高柳。

高柳回了一封。不是单独给林奕的——是群发。

`如果 commit 密度有周期性——那说明系统本身在按某种节律运行。不是林奕的操作触发了记录。是系统在按自己的节奏观测世界线。林奕的操作是观测窗口里的噪声。`

林奕看到邮件的时候在便利店。

系统在按自己的节奏观测。

不是他触发了记录。是观测本来就存在。他的渗透只是在观测窗口里留下了痕迹。

他忽然想——那 kiri 的 commit 也是这个周期的一部分吗。

他不知道。

但他知道有人可能知道。

他打开论坛私信。kagure 的头像是灰色的。他打了一行字，删掉。又打了一行，又删掉。

最后他发了一句：

`你还在吗。`

他等了一会儿。没有回复。他把手机放进口袋。

便利店外面。十一月的阳光是冷的。千夏还在做题。
