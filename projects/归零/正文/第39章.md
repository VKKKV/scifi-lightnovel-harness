# 第39章 · 回声

凌没有离开秋叶原那个网咖。林奕去找他的时候，他还坐在同一个位置。三台笔记本。烟灰缸满了。外卖盒堆在桌角。

「你没睡。」

「你也是。」

林奕把 P53 放在桌上。凌看了一眼这台电脑——A 面的刻字。他没有再问关于遥的事。那句话说完了。

「叶岚发了一封邮件。关于 hex tree 的深层加密层。她说她找到了入口——但需要渗透经验来突破。她管这叫外部观测。你需要做内部操作。」

凌没有回答。他打开一台笔记本，屏幕上是一个 git log 的界面——不是 wl0 的。是他自己的。他也在用 git 记录。但记录的不是世界线——是他自己的追查经过。六年的 commit log。

他翻到最近一条。作者：kagure。日期：今天。内容：一个 IP 地址和一个时间戳。

「你那条 observer-external 的 anomaly 出现的时候——我的机器也被扫了。」凌说。「不是你的系统在追踪我。是有人在通过你的系统追踪每一个连接到 wl0 的人。」

「你断网了。」

「断网没有用。」凌把屏幕转过来。「他们没有用网络找。他们用物理位置。」

屏幕上是一张地图。秋叶原。一个红点——凌的位置。标注的时间是他拔网线之后的五分钟。

林奕沉默了。

「你还在用那个网咖？」

「换了一家。」

「有用吗。」

「不知道。」

***

晚上。叶岚、凌、林奕在同一个聊天窗口里。不是论坛私信——是 IRC。叶岚建的频道。三个人各自在不同的地方。

叶岚发了一张图——hex tree 深层文件的结构分析。她标出了加密层的类型——不是对称加密。不是非对称。是一种她没有见过的混合结构。

凌看了图之后打了几个字：

`这不是人的设计。这不是加密——是协议。`

`什么意思。`

`你在和系统握手。每次读取深层文件——都是在向系统发送请求。响应不是文件内容——是系统对你的权限评估。`

叶岚沉默了。

林奕看着那行字。

凌继续说：

`遥的笔记里写过一句话。我之前不理解：""深层文件不需要破解。需要权限。""她说的不是计算机权限。是某种——存在权限。系统判定你有没有资格看。`

林奕问：

`怎么提升权限。`

凌的回复过了很久才出现：

`不知道。她没写。`

***

凌晨。林奕从 IRC 下线。他打开 P53。没有渗透。没有翻 log。他看着 A 面的刻字。ハルカ と リョウ。

在秋叶原那个积满烟灰的桌面上，凌说「她给我做的。我从来不知道」的时候——声音很平。不是冷。是把所有情绪压到一条直线上，不让它弯曲。

林奕想——如果六年后有人告诉我，我身边的人留给我的东西在某个角落等了我六年——我不知道自己能不能用那么平的声音说话。

他把 P53 的 A 面转过去。打开屏幕。输入了一行命令——不是 git log。是 `ls`。hex tree 根目录下的文件列表。

他一个个看。不是找什么。是想记住。每一个目录。每一个名字。它们是美佐留下的。是遥触碰过的。是凌追了六年才在另一个人的电脑上看到的。

他关了 P53。

窗外月亮很细。明天还要去便利店。
