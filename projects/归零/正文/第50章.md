# 第50章 · 死锁

林奕知道谦治的死因——器官捐献在手术前被组织截下。他知道了因果链的起点。他想把它改掉。

他在 git log 里找到了目标时间点。通过 wl0 隧道连接。目标：医院器官运输系统的调度记录。他不需要黑进组织——他只需要改调度系统的路线，让器官提前一班车到达。组织截不到。谦治活。

操作很简单。比 Modbus 简单。只改一条记录。

他做了。

渗透成功。器官运输路线改了。组织没有截到。器官到达医院。谦治活。

然后现实开始更新。

不是渐变的。是瞬间的。他坐在 P53 前——屏幕上的终端输出扭曲了一下。像是信号中断了半秒。

然后一切都变了。

谦治活了。他没有在手术台上死。他继续经营浅羽商店。他更谨慎地处理服务器——因为他活着，他知道服务器上有美佐留下的东西。他不会让它落入任何人手里。

不卖 P53。不让人进仓库。不信任任何人。

P53 没有出现在杂货店的货架上。

林奕没有买到 P53。

wl0 没有触发。

林奕坐在公寓里。他用的是自己的旧笔记本。桌面上没有黑窗口。没有虚拟网卡。没有 git log。一切都正常——太正常了。正常到他无法呼吸。

他救活了谦治。但代价是——他从未获得终端。从未救过千夏。

千夏——她活在那条谦治还活着的世界线上。但林奕不知道她是否被卡车撞过。因为那个时间点他没有在最渗透去救她。他没有 wl0。他什么都不是——只是一个普通留学生。

他重新打开笔记本。没有黑窗口。没有 git log。没有 daemon.md。一切像从来没发生过。

他坐在桌前。手放在键盘上。屏幕亮着。但没有任何东西在等他了。

现实开始回滚。

不是他操作的——是系统。因果死锁。谦治活了 → P53 没卖 → 林奕没买 → 没有终端 → 没有渗透 → 谦治还是死了。

世界线绷了一下。然后弹回来了。

林奕睁开眼睛。P53 在面前。终端开着。谦治死了。器官被截了。和之前一样。

他救不了谦治。

他坐在 P53 前。屏幕的光照亮了他的脸。他刚刚经历了一次世界线回滚。他记得一切——谦治活下来的那条线。他自己的不存在。回滚时的那种失重感。

他在 daemon-v2.md 里打了一行字：

`谦治的死是我能买到 P53 的前提条件。我不能救他。`

他把那行字看了很久。然后没有加 `——但他不该死。` 因为加也没有用。
