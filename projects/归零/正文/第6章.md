# 第6章 · 边界

千夏的计算机课从一张杂货店的收银小票背面开始。

林奕拿了她放在柜台上的圆珠笔——就是那支咬过的——把小票翻过来，在背面画了一个矩形。

「这是终端。黑窗口。你现在看到的是什么。」

千夏低头看了一眼。她今天没扎头发，散着的，垂下来遮住了半张脸。她把头发别到耳朵后面——动作很快，像是不喜欢头发挡视线。

「黑的。左上角有个绿色的光标。旁边有字——」

「那是提示符。不用管它。你现在能做什么。」

「不知道。我敲过键盘。出来的字我不知道是什么意思。」

「你敲了什么。」

千夏想了想。「ls。」

林奕挑了下眉毛——不是意外，是「你居然知道这个」的表情。「你搜过。」

「搜了。但搜完不知道对不对。因为出来的是英文——」

「英文你可以看。」

「不是英文的问题。」她从他手里把笔拿回去——不是抢，是自然地拿回去，像那个笔本来就不该在别人手里。她在小票上写了一个词。Linux。「我搜这个。出来了……很多。几个小时看不完。然后我搜『终端怎么用』——出来的教我怎么开窗口。但我已经开了。我不知道怎么搜下一步。」

林奕懂了。

「再来。」林奕把笔放回柜台上。「你敲 `ls`。回车。看到了什么。」

「几个词。蓝色和绿色的。」

「那是文件和目录。蓝色的——」

两个人头凑在柜台上方。小票上的矩形旁边又多了一行字——`cd`。林奕写得很小，因为小票没地方了。千夏凑近了看，头发又垂下来了。这次她没别——直接用手拢着，手指按在耳朵后面。她另一只手在柜台下面的本子上写着什么——不是小票上的笔记。是她的笔记本。她之前一直往上记服务器指示灯和硬盘声音的那本。现在在记 `ls` 和 `cd`。

下课的时候林奕问了一句：「你父亲的电脑——在上面？」

千夏点头。她没抬头，还在本子上写字。「二楼。一直开着。我不碰它——怕弄坏。但现在我想知道上面有什么。」

***

当天晚上，林奕回到公寓。

赵逸风不在——他去秋叶原了，说是有限定手办发售，不排队抢不到。公寓里只有 P53 屏幕的光。窗外路灯亮着。今天晚上没有管人直播的声音——安静得有点空。

林奕打开终端。`git log` 又多了一条 commit——不是他操作的。是自动的。他翻了一下——内容是他下午在杂货店和千夏在柜台后面说话的整个时间段。网络流量为空——没有 HTTP 请求，没有 ARP。就只是一个时间区间被记录了。commit message 是一串 hex，解码之后是一个英文单词：`observed`。

被观测了。

林奕盯着那行字。不是他主动记录的任何东西——是这台机器自己在看。在记。他和千夏在柜台后面上了两个小时课。P53 在公寓里。但它记下来了。

他把这个记进 daemon.md。

`第二条：系统会自己记录东西。不只是我做的操作。是发生的事。`

然后他开始计划第二次渗透。

千夏活着。他每天都能确认。这次是为了别的。第一次渗透他改了监控时间戳——代价是三天后他发现自己多了一张超市会员卡。卡上印着他的名字，积分累计日期从三个月前开始。他不记得去过那家超市。但他钱包里有那张卡。

蝴蝶效应？

他不知道。但他需要更多数据。

这次他选了一个更小的目标——小到几乎不会产生连锁反应。他在 git log 里翻到了三天前的一条记录：某个他常逛的技术论坛，有人发了一篇帖子骂 Arch 的包管理——里面附了发帖人的 IP、时间戳和完整的 HTTP 请求。他通过 wl0 隧道连到那个时间点的论坛服务器。这次不用漏洞——论坛后台的测试账号还在，密码 `test123`。进去。找到那篇帖子。删了。

整个操作不到三分钟。

git log 多了一条 commit。daemon.md 第三条记录。

然后他开始等——等现实中有什么会变。

第一天。没有。第二天。没有。第三天他以为这次逃过了——然后赵逸风在吃泡面的时候突然说了一句话。

「诶，你还记得之前那个在论坛上骂 Arch 的人吗。」

林奕的筷子停住了。

「他那个帖子写得挺搞笑的——你怎么了。」

「你还记得那个帖子。」

「记得啊。就前两天看到的——他说 Arch 的依赖关系像意大利面。然后下面有人喷他——」

那个帖子被删了。林奕亲手删的。在它发布的那天——三天前。赵逸风不该看到。他不逛技术论坛。他连 Arch 是什么都不知道。

但他说他看到了。

林奕放下筷子。他把碗推到一边，打开电脑。搜论坛。那篇帖子已经不存在了——他删了。但在搜索引擎缓存里，有页面的快照。点开——帖子还在。下面的评论也全在。其中一条评论是他自己发的——他不记得发过。语气和他一模一样。内容是「你装的是 Manjaro 不是 Arch」。他从来没有在这个帖子里回过任何东西。

但在赵逸风的记忆里——他看过这个帖子。在这个世界上——林奕回过这个帖子。

他不知道怎么解释。帖子删了。但赵逸风记得。他自己回了那条评论——他不记得回过。好像删掉的东西没有消失，只是换了一个人补上。赵逸风变成了知情者。

林奕盯着屏幕上的快照。他回的那条评论下面又多了一条新的——一个他不认识的头像。内容只有一个词：

`interesting.`

发布时间是他删帖的同一秒。

他把这条记录写进 daemon.md。

`第三条：删了一个帖子。室友多了一段不该有的记忆。有人在看我。`
