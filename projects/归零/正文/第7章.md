# 第7章 · 规则

林奕花了一周把 daemon.md 从三条扩展到了十七条。

不是一口气写的。是每次渗透之后记一条——有时候记的不是规则，是猜测。猜测后面打个问号，下次渗透如果验证了就删掉问号。如果推翻就整行删掉重写。

daemon.md 上的规则越来越多。每一条都是试出来的。

第四、五、六次渗透他选的都是无关紧要的东西。

补发一封三天前被邮箱过滤器拦下来的邮件——一个 BBS 管理员发给另一个管理员的换版主通知。不是敏感信息。不是能改变任何重大事件的东西。他只是想看看：最小单位的渗透会产生多大范围的蝴蝶效应。

邮件发出去了。第二天他发现 BBS 的版主名单变了——不是邮件里提到的那两个人。是和他们有关的第三个人。一个他从来没搜过、从来没碰过的人。被换掉了。因为在新的因果链里，那封邮件被另一个人先看到了——那个人在换版主讨论串里投了关键一票。

林奕在 daemon.md 里加了一条。

`规则 4：渗透的连锁反应通过信息传播扩散。一封邮件不只是收件人看到。每一个转发、每一次提及、每一次「我听说」——都是新的因果分支。`

第五条渗透他试了更远的时间。不是几天前——是两年前。他在 git log 里翻到了一个旧的网络事件：某大学服务器被入侵的日志，攻击者用了 SQL 注入，手法粗糙，是初学者。林奕通过 wl0 连到那个时间点，在攻击者的 shell 里加了一行防火墙规则——把他的 IP 封了。攻击没成功。服务器没被入侵。

回到现在。那所大学的计算机社团招生多了三个人。因为服务器没被入侵的那个时间线上，社团的网站没挂——多挂了一天，正好被一个高中生看到。那个高中生就是现在多出来的三个新生之一。

`规则 5：时间越远，连锁反应越广。但方向不可控。`

第六条渗透他又回到三天内的范围。改了一篇技术博客的发布时间——让它在搜索引擎里排得更靠前。效果范围很小。蝴蝶效应几乎没有——只有博客作者多了一个 Twitter 关注者。

`规则 6（推测）：时间距离越近，波及范围越小。时间越远，波及越广。正向关系——但不是线性。`

他把这行标了问号。

窗外是晚上十一点。赵逸风刚从便利店回来——不是拐角那家，是更远的罗森，因为拐角那家没有他想要的限定口味薯片。他把袋子扔在桌上，看了一眼林奕的屏幕。

「你这黑窗口怎么一直在闪。」

林奕把终端最小化了。不是想藏——是赵逸风问的话越来越难搪塞。上次他说「这是编译器」，赵逸风信了。但他不可能永远信。

「你不是去抢限定手办了吗。」

「抢到了。」赵逸风从袋子里掏出一个盒子——透明的塑料壳，里面是一个拿着吉他、闭着眼睛的女角色。他把盒子放在自己桌上，和三块屏幕并排。「最后一盒。我后面那个人差点打我。」

「为了一个塑料。」

「这不是塑料。这是限定。」赵逸风坐在椅子上，转了半圈。「你今天又没出门。」

林奕没回答。他这一周出门的次数确实比之前少——除了去杂货店帮千夏搬货和上课，他大部分时间都在 P53 前。不是不想出门。是每次渗透之后他需要观察——看现实中多了什么、少了什么、变了什么。

赵逸风没追问。他撕开限定薯片的包装袋——是海苔味。房间里弥漫着海苔和旧纸箱混合的味道。他放了一个 Vtuber 的直播回放，声音开得很低。

林奕把终端切回来。屏幕上 `git log` 又多了一条——又是自动的。commit message 解码之后是 `observed`。和上次一样。

系统在看他。

`observed`。每一条都是。他在公寓里吃泡面、出去买绿茶、在杂货店搬货、和千夏讲 `cd` 和 `ls` 的区别——这些全在 git log 里。时间戳精确到秒。

他试过关掉 P53。`observed` 的 commit 没有断——在关机期间没有记录。但开机之后——立刻补了关机时间段的一条 commit。不是空白。是一段 hex 编码的状态标记——解码之后是一个词：`idle`。不是睡眠。是进程状态。

系统知道他睡着了。

林奕在 daemon.md 里加了一条。

`规则 7：系统会观测一切。不仅是渗透。是持有者的全部行为。包括睡眠。`

他盯着这行字看了一会儿。然后又在下面加了一行。

`谁在看我？`

问号。

***

第二天下午，林奕去杂货店搬货。

翔太不在——他去排练了，说是下个月有场重要的演出，不能丢人。千夏一个人在店里。货架上的旧杂志换了一批新的——不是新的内容，是把旧的翻了个面，封面朝外。千夏说这样看起来新鲜。林奕没说这没用。

充电宝卖得不错，架子上空了一半。千夏从仓库搬了新的一箱出来——这次不是充电宝，是手机数据线。她拆箱的时候美工刀划得比上次稳——已经知道自己要用多大力了。

搬完货之后上课。这次不在柜台——千夏把林奕带到二楼。

楼梯很窄，每级的高度不一样——旧建筑，地板有点斜。二楼是一个小房间，不大。一张桌子。一台机架式服务器立在墙角——两米高，黑色的，指示灯在暗光里闪绿色和蓝色。风扇的声音很轻——不是安静的轻，是那种久了你就忘了它还在响的轻。

服务器旁边的桌上放着一台旧显示器和键盘。Dell，屏幕上有划痕。键盘是 PS/2 接口的——不是 USB。

「这就是我爸的电脑。」千夏站在门口说。她没有走进去——像那个房间有一个她不该碰的边界。「我从来没关过它。」

林奕看着服务器上闪烁的指示灯。机器很老了——不是配置老，是年代老，那种你不确定它在跑什么、但你知道它一直在跑的机器。

「你知道上面有什么吗。」

「不知道。」千夏靠在门框上，视线停在服务器上。「我爸说别让它停下来。所以我不关。」

「你从来没上去看过。」

「不敢。」她停了一下。「怕弄坏。」

林奕走到服务器前面。屏幕是亮着的——终端界面。和他在 P53 上的一样。黑色背景，绿色光标。最上面一行写着系统信息和上次登录时间。上次登录是三年前。

有人三年没碰过这台机器。但它一直在跑。

「你想看吗。」林奕问。

千夏走进来了。第一次。脚步很慢——不是犹豫，是那种进入一个你一直想进但没敢进的房间时，身体不自觉的谨慎。她站在林奕旁边。屏幕上的光标在闪。服务器的风扇声填满了沉默。

「我不知道从哪开始。」她说。

「你已经开始了。」林奕指着终端上的光标。「你知道怎么让它等你。」

千夏看着光标。然后把手放在键盘上。

没敲。只是放着。

「下次。」她说。

她把键盘推回去。不是放弃——是把这一刻留到她想好了要敲什么的时候。

林奕看着她退出房间。她在楼梯口停了一下，回头看了一眼服务器的指示灯。绿色的还在闪。蓝色的也在闪。

然后她下楼了。
