# 第42章 · 意外

第一件事是赵逸风。

他在回公寓的路上经过一栋施工中的大楼——不是他平时走的路。是那天他为了买限定薯片绕了一段。一块金属板从脚手架边缘滑下来。砸在他身后三米的位置。

声音很大。路上的人都在看。

赵逸风回头看着那块板。地上砸了一个坑。他站了一会儿。绕开。继续走。

回到公寓之后他跟林奕说了一句话：「今天差点被砸死。」

林奕正在翻 git log。手指停了。

「在哪。」

赵逸风说了地址。

林奕没有说那栋楼不是赵逸风平时走的路。没有说蝴蝶效应。没有说任何事。他打开了 git log——搜索那个时间段的 commit。有一条新的。不是他操作的。

commit message 只有一个字：`挡`。

作者栏是空的。

***

第二件事是叶岚的实验室断电。

不是整栋楼——只有她那一层的电。备份服务器全部离线。三天的工作丢失。不是计算机的停电——是「电箱跳闸了」。管理员说是老旧线路的问题。叶岚检查了电箱——跳闸的开关上有一个指印。不是管理员的。

她没有报警。她回到实验室，用电池给笔记本供电，继续工作。

凌晨她在 IRC 里说了一句：

`有人进过实验室。`

凌问：

`丢什么了。`

`没丢。只是翻了。`

***

第三件事是便利店门口的车。

一辆白色轿车。没有车牌。停在便利店门口的路边。每天早上位置都不一样——往前移几米、往后移几米。千夏第三天注意到的时候，在本子上记了一个时间。

「门口那辆车。第三天。早上六点零二分在。下午三点不在。晚上七点又在。」

林奕站在柜台后面往外看。车窗是黑的。看不见里面。

他没有告诉千夏不要出去。

***

第四件事是无人机。

高柳在回家路上发现头顶有一个小型的无人机——飞得很高。不是玩具。没有指示灯。他在路灯亮起来的时候看到了它的轮廓。他用手机拍了照。放大了看——机身没有任何标识。

他改道走了一条更亮的街。无人机跟着。走了半条街——然后消失了。不是降落了。是关灯了。在黑暗里继续跟。

高柳那天多走了两站路才回家。

***

林奕在 IRC 频道里发了那条「挡」的 commit 截图。

`这不是系统操作。author 是空的。但时间戳是赵逸风差点被砸的那一刻。`

凌过了很久才回复：

`有人在你之前操作了系统。不是渗透——是用别的方式。`

`什么方式。`

`我不知道。但我知道一件事——` 凌正在输入。停了。继续。 `——有人在保护你们。代价是暴露自己。`

林奕看着那行字。

B1 店主的抬头。旧夹克的领口。擦杯子的手。

「不是第一次被带进这里。」

他没有把这句话发到 IRC。

但他知道是谁挡的那一下。
