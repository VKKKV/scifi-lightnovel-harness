# 世界观

归零的世界观建立在「终端 = 时空观测器」的核心设定上。
详细约束见 `.harness-project/constraints.md` R-G 系列。

## 关键地点

- 东京千代田区：林奕合租公寓、语校
- 浅羽商店/便利店：千夏家的杂货店（一楼便利店 + 二楼服务器）
- B1：地下 livehouse，高柳常去，店主是沉默老头（组织前成员）
- 高円寺 HIGH：翔太演出的 livehouse
- 神保町 Asaba 電脳：买 P53 的地方

## 终端与服务器

- P53（林奕的终端）：ThinkPad P53，wl0 网卡，世界线 1919810
- 浅羽谦治遗留服务器：便利店二楼，hex tree 结构
- wl0：虚拟网卡，连接过去时间点网络的隧道
- git log：自动记录所有观测和操作
