# 背景设定

## 时代

202X 年，当代。

## 地点

日本东京。具体场景包括：
- 林奕和赵逸风合租的公寓
- 语校
- 千夏的便利店（父亲留下的 BBS 时代电脑店改建）
- 秋叶原
- 线上：Arch Linux 论坛、IRC、CTF 平台

## 世界观概要

### 三层结构

1. **表层**：普通日本都市生活（语校、便利店、秋叶原、合租日常）
2. **里层**：地下黑客社区（Arch Linux 论坛、CTF 圈子、黑客榜单）
3. **深层**：世界线系统、接触过终端的少数人、kiri、幕后组织

### 虚拟网卡 wl0

- 在 CTF 中误操作后被挂载到林奕的机器上
- `wl0` = world-line 0
- 通过 `ssh root@worldline.{编号}` 连接世界线
- 当前世界线编号：1919810

### 世界线系统

- 终端界面呈现为类 Unix 风格——这是科学家团队设计的用户界面
- 底层是一台超级量子计算机——几十年前由科学家团队在幕后组织资助下建造。拥有超级 AI，能推演过去未来、干涉因果律
- 实验失控后，良心发现的科学家（美佐是其中之一）人为制造爆炸。量子计算机命运不明。终端变成了一种不确定的状态
- kagure 用 git 术语（branch / reset / commit / log）作为教学隐喻向林奕解释
- 高柳教授从量子信息角度提供理论框架
- 林奕在第一卷中相信 git 就是真相；到了第二卷才开始怀疑这套比喻不够用

- 类似 wechall 的在线黑客排行系统
- 日本榜：#1 kagure，#2 waterlily（已数年未登录，实为同一人）
- 中国榜：#1 未知 ID（kiri），#2 顾惟清（北理工教授），#3 叶岚

### 千夏父亲的服务器

- 年代：BBS 时代延续至今，CentOS 5，内核 2.6
- 只开放 telnet 和 SSH
- 隐藏目录：美佐留下的 hex tree——文件名和目录名都是 16 进制乱码。只有终端持有者能在 P53 上解码阅读
- 功能：美佐将终端相关知识编码在此。谦治知道它的存在但从未打开
