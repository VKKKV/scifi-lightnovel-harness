# 核心设定

## 书名

归零

## 类型

科幻悬疑 + 都市日常 + 黑客/计算机文化

## 核心概念

虚拟网卡 wl0 = 时空 VPN 隧道。通过 wl0 连接过去时间点的网络，进行 CTF 渗透——改变过去。

git log = 观测面板（只读）。每笔 commit 自动记录世界线变化。

daemon.md = 林奕手动记录的规则笔记本，通过试错推导操作规律。

## 代价：蝴蝶效应

- 每次渗透 → 历史从修改点重新计算 → 连锁反应超出预测
- 渗透规模越大 → 波及范围越广 → 林奕周围的人被牵连
- 不是系统惩罚——是因果律没有安全边界

## 闪回：déjà vu

被抹除的世界线分支留下闪回碎片——déjà vu，不是完整记忆。
大脑只是辅助信号——大部分操作知识靠 git log 和笔记。

## 规则约束

1. 终端不拟人——没有任何提示/消息/警告。林奕自己推导一切。
2. 唯一操作 = CTF 渗透。git 命令只读。
3. 蝴蝶效应有因果链，不是随机厄运。
4. 闪回 ≠ 完整记忆（终局回溯例外）。
5. 同一时刻只有一人持有终端权限。
6. 量子术语只从高柳教授和 kiri 视角给出。
7. 唯一奇幻：终端为何选中林奕——不解释。

## 四卷结构

| 卷 | 标题 | 核心 | 渗透层级 |
|:---|:-----|:-----|:---------|
| 一 | 连接 | 日常建立/能力发现/全员登场/首次蝴蝶效应/团队摊牌 | 初级→中级入门 |
| 二 | 提权 | 能力深化/服务器探索/kiri浮现/凌交汇引爆 | 中级→高级入门 |

## 叙事规范

- 第三人称限知视角（以林奕为主，偶尔切千夏/凌）
- 正文不用日语（专有名词/刻字/文化概念例外）
- 情感用控制写：越大情感 → 越小动作
- 日常主线比例 ~50/50
