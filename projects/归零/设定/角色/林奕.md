## 金手指：wl0 时空 VPN 隧道

虚拟网卡 `wl0` 的本质是一个能连接并代理到**过去时间点计算机网络**的时空隧道。类 Unix 终端只是用户界面，不是系统本质。

- `ssh root@worldline.{编号}` → 连接目标世界线的网络环境
- IP 地址中的数字编码时间偏移量（如 `192.168.114.1.114514`）
- git 命令（log/reset/branch）只是观测面板——查事件快照、回退观测焦点、标记路径。不能直接修改现实

**唯一操作方式 = 跨时空黑客渗透**：
```
git log 发现目标 → 定位目标 IP（含时间坐标）→ wl0 隧道连接
→ CTF 渗透攻击（SQLi/Pwn/Rev）→ 拿到 shell → 修改目标数据
→ 历史被改写 → 现实即时更新
```

**能力分层**：初级（Web/BBS 渗透）→ 中级（医院/交通系统）→ 高级（核心基础设施）
**核心限制**：每次渗透产生蝴蝶效应——连锁反应超出预测。渗透越大 → 波及越广 → 林奕周围的人被牵连
**唯一持有者**：同一时刻仅一人。只有林奕——他是第一个，没有前任。
