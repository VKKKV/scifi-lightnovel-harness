# 神楽 遥（Kagura Haruka）— 已故 / 宫城凌的恋人 / P53 原本的主人

## 定位

已故人物 / 宫城凌的情感锚点 / P53 的来源 / 未送达的 Commit

## 基本信息

- 姓名：神楽 遥（Kagura Haruka）
- 网络 ID：`kagure`（神楽）
- A 面刻字：`ハルカ`（片假名）
- 状态：已故（2019 年末。组织制造的「意外」——和她姐姐美佐一样）
- 生前身份：安全工程师，CTF 选手，Arch Linux 活跃贡献者
- 平台/排行：日本榜第一
- 家族：浅羽美佐（千夏母亲）的妹妹——千夏的姨妈。美佐嫁给谦治后，遥通过姐姐认识了谦治的电脑店

## 与宫城凌（リョウ）的关系

- 宫城凌（ID: `waterlily`、名：凌 / リョウ）的恋人
- 两人通过 CTF 和 Arch Linux 社区相识——遥是社区核心维护者之一
- 凌是对算力有极度狂热的 Arch 玩家。遥知道这件事
- 2019 年，遥在谦治的电脑店花重金定制了一台 P53 顶配工作站——128GB 内存、12TB 硬盘、Quadro RTX 5000——作为送给凌的生日礼物。她找谦治不是偶然——谦治是她姐夫
- 她要求在 A 面（顶盖）右下角用银色细线体刻下两人的名：`ハルカ と リョウ`
- **凌从始至终不知道这台电脑的存在**

## 与浅羽美佐的关系

- 美佐是遥的姐姐。当美佐出逃、嫁给谦治以求安全时——遥知道姐姐在躲什么。美佐没有告诉她全部——但遥自己也是安全工程师，她能从碎片中推断姐姐在保护一个量子层级的秘密
- 美佐死后，遥继承了姐姐留下的一部分观测记录——不是终端权限，是美佐用自己的方式记录的观测日志。遥从未持有终端，但她知道终端的存在
- 遥之死（2019 年末）——组织制造的「意外」。和姐姐美佐一样。凌不知道是谋杀——他只知道她突然不在了

## P53 的流转（「未送达的 Commit」）

1. **2019 年**：遥在谦治的店下单 P53 → 谦治亲手组装、调优、刻字 → 尚未交付
2. **2019 年末**：遥死亡 → P53 成为一件没有收件人的包裹
3. **~2020 年**：谦治将 P53 按原地址寄给凌 → 凌因女友死亡闭门不出 → 无人签收 → 快递退回
4. **~2020-2021 年**：谦治被世界意识消除。P53 留在仓库——一台 128GB 的猛兽，装在没有字的纸箱里，落灰
5. **现在**：千夏清理货架。翔太翻出这台「开机黑屏、没电池」的旧机器，按回收价卖了。买的人是——林奕

凌从始至终不知道。六年后，这台机器将在一个昏暗的 livehouse 里——让他停住。

## 宫城凌的追查（新设）

**凌从未持有终端权限。** 遥死后，凌在整理她的遗物时发现了线索——不是终端本身，是她留下的笔记和日志碎片。凌凭这些线索独自推断出了一部分真相。他换掉 waterlily 的 ID，改用 kagure——不是技术选择，是他想用她的身份活在她来过的地方。

他的追查至今没有结束。

## 性格（通过遗物和他人叙述呈现）

- 技术狂——凌后来提起她时总是先说「她的代码」，然后沉默
- 比凌更沉默寡言，但会在关键时刻说一句让人忘不掉的话
- 签名风格和凌一模一样——实际上凌的很多习惯都是从她那里学来的
- 谦治留下的工作笔记里提到她：「遥さん。あの子はコードを書くために生まれてきた。僕はそれを箱に詰めるだけ。」（遥小姐。那孩子是为了写代码而出生的。我只是把它装进箱子而已。）——「箱子」就是那台 P53。谦治知道这台机器的意义。他没有告诉凌。因为他也知道——已经没有必要了

## 叙事功能

- 凌的情感核心——解释他的换 ID、为什么帮林奕
- P53 的来源锚点——A 面刻字是全书的隐藏伏笔
- **终极引爆**：凌在 livehouse/线下接头处看到 A 面刻字——这是他第一次知道这台电脑的存在。他六年来拼命想要忘记的过去，他以为早就被埋掉的爱意——以硬件实体的形式，跨越时间送到了他面前

## A 面刻字方案

- 位置：A 面（顶盖）右下角
- 字体：银色细线体，机器刻字
- 内容：`ハルカ と リョウ`
- 只有名（不是姓）——信息隐藏的极佳设计。林奕读得出读音，不认识对应的汉字和人物
- **伏笔错位**：林奕在论坛上看到 ID `kagure`（神楽）时——不会立刻联想到「ハルカ」——因为一个是姓，一个是名。同一个人的身份被拆分在了两个不同的信息层里
- 老赵的连接：后期某天，老赵无意中看到 A 面刻字——「ハルカ？这个发音怎么感觉在哪听过的——」他不是计算机圈的人，联想不到 hacker 排行。但他的二次元脑对名字的直觉——可能会比林奕先踩到那个点上
