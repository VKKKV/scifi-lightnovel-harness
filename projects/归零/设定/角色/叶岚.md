# 叶岚 — 女主 2

## 定位

技术天才 / 林奕的镜像 → 核心团队成员 / 第二个靠自己推导出终端存在的人

## 基本信息

- 姓名：叶岚（Ye Lan）
- 年龄：18-20 岁（与林奕相仿）
- 性别：女
- 身份：中国人，东工大计算机系本硕博连读

## 家庭背景

- 来东京的核心原因：逃离原生家庭，心底又舍不得父母
- 中学时期因「只擅长计算机的 nerd」身份遭遇校园霸凌
- 霸凌最严重时休学在家，不接受任何来信和信息
- 就是这段封闭期，冲上了黑客排行榜中国第三

## 学术路径

- 霸凌最严重时休学在家，不接受任何来信和信息
- 就是这段封闭期，冲上了黑客排行榜中国第三
- 大学里的教授是她加密项目的代码贡献者之一，与她线上相识后邀请她来日本
- 教授本想让她高中就来，但当时正值休学期，她断绝了所有联系
- 恢复通信后，教授重新发出邀请——本硕博连读

## 技术层级

- 计算机网络大神 + 密码学博士级水准
- 维护着类似 GitHub 平台上 star 最高的综合流量加密内核项目（类似 xraycore 但更强）
- 另有多项自研加密算法
- 黑客排行榜：中国 #3（#2 北理工某教授，本科时期冲上榜至今仍活跃；#1 未知 ID：`kiri`）
- 编辑器：VSCode。看到林奕的 Vim + Colemak 会觉得他在浪费时间

## 性格

- 沉默寡言，但每句都扎在关键处
- 自视甚高——认为「自己只是做到了任何人付出一点点努力就能做到的事情」
- 也正因为这种认知，有点瞧不起折腾 Vim 和 Arch Linux 的林奕
- 在被需要的时候有队长气质
- 和她线下讨论技术时完全是另一个人

## 与林奕的关系

- CTF 决赛指点相识：林奕在 Wireshark 上卡了一个小时，她路过扫了一眼：「你 filter 写错了」
- 逐渐成为队友——远程配合默契到两人自己都觉得吓人
- 他被叶岚认为是「行为艺术」的技术习惯（Vim + Colemak）成为两人日常互怼的素材

## 与千夏的关系

- 叶岚是主角团里唯一能回答千夏密码学问题的人。千夏问她，她就给方向——不直接给答案
- 两人之间不是谁教谁——千夏问问题，叶岚说「你自己搜」，千夏搜完再回来聊。来来回回就熟了
- 她们互补：一个懂机器，一个懂人

## 独立行动线

- 纯学术方法（网络测量、流量分析）追踪世界线异常痕迹
- 自己推断出了世界线的 DAG 结构碎片——但缺少最核心的一块拼图
- 完整真相来自林奕——不是她没推出来，是林奕选择说出来

## 知道终端之后

- 林奕在蝴蝶效应失控后，第一次选择求助——把团队聚在一起，摊开了终端的事
- 千夏、叶岚、高柳——同一天知道的
- 叶岚的反应：没有恐慌。一个做了多年密码学的人听到世界线有 Unix 面板——第一反应是「有意思」
- 从镜像变成互补：林奕暴力渗透，她用学术方法分析规律
