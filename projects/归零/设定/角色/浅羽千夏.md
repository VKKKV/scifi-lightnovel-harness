# 浅羽千夏（Asaba Chinatsu）

## 定位

情感锚点 / 便利店店主 / 独立推导者（与林奕镜像）

## 基本信息

- 姓名：浅羽千夏
- 年龄：20 岁
- 性别：女
- 身份：东京本地大学生。空闲时间在自家杂货店度过。父亲遗产够生活，坚持开店因为「关了店就等于承认他不在了。」

## Want / Need / Flaw

- **Want（她想要的）**：守住父亲的店。不只是「不想让它关门」——是「如果我关了它，就等于承认他不在了。」
- **Need（她需要的，但她不知道）**：理解父亲最后那句话——「别让这台服务器停下来。」不是把它当遗物供着——是知道它为什么重要。
- **Flaw（缺陷）**：她相信父亲的遗言是一个必须原样执行的命令——不需要理解，只需要服从。不问「为什么」——因为在她的认知里，问了就等于不信任。她一直没碰母亲的笔记本也是因为这个。

## 成长弧线（独立于林奕）

```
起点：守护，不追问
  ↓
触发：服务器出现她解释不了的异常（她自己发现的）
  ↓
转折：她决定不再只是「守着」——开始自己学计算机。不是因为林奕教她，是因为她想知道父亲在保护什么
  ↓
第一卷末：林奕在 B1 摊牌——千夏第一次知道服务器是什么。不是惊喜——是确认。她一直在守护的东西——终于有人告诉她那是什么了
  ↓
终点：当她最终知道全部真相时，她不是被动得知的——是她自己一步步挖到的
```

## 独立行动线

- **自己发现服务器异常**：千夏每天都会上楼看一眼服务器的指示灯。某天她注意到了一个之前没见过的东西——不是林奕告诉她的。她自己看到的。
- **自己决定学计算机**：林奕只是教她——推她决定学的是她自己。「我想看懂这行日志」——不是「你教我这行日志」。
- 自己的笔记：她在柜台后面有一个本子，记录服务器上的变化——指示灯的颜色、硬盘读写的声音模式。和林奕的 daemon.md 是镜像——两个人在两台不同的终端前，用完全不同的方式，记录同一个系统。
- **比林奕更早感知异常**：因为她每天都在看同一台服务器——某天她注意到的变化，林奕的终端上还没出现。

## 与林奕的相识

- 林奕通过谷歌地图（信息未更新，仍显示电脑店）来到杂货店。翔太看店时卖出 P53。千夏在门口擦肩。
- livehouse 翔太堵人后，林奕来打工还差价 → 正式认识。

## 学计算机的方式

- 初期跟着林奕学——终端怎么打开、命令怎么敲。朋友之间随手教的，不是上课
- 独自发现服务器上有加密文件后，对密码学产生兴趣。开始自己琢磨——不是为了学技术，是为了打开那个文件
- 和叶岚熟了以后，千夏有密码学的问题会问她。叶岚的教法就是给方向：「你自己搜。搜不到再来问我。」千夏也习惯了

## 父亲：浅羽谦治

- BBS 时代电脑店店主。几年前去世——表层原因写的是肝功能衰竭等不到器官捐献，真正原因：捐献者的器官被组织截下。
- **父亲留下的服务器**：一台 2 米高机架。只开 telnet 和 SSH（仅限本地登录）。服务器本身没有特殊能力——只是谦治生前维护的旧机器。千夏守着它，因为那是父亲最后的话：「别让这台服务器停下来。」
- 最后的话：「别让这台服务器停下来。」不是沉重的负担——不断电不断网，服务器自己就能跑。
- 服务器上的加密文件——美佐留下的 hex tree——才是真正的秘密。谦治可能知道它的存在，但从未告诉千夏。

## 母亲：浅羽美佐

- 已故。千夏对她只有碎片记忆——以及父亲从不提起她的事实。
- 真相（千夏不知道）：美佐是全球顶尖科学家团队的成员。团队建造了终端的实验原型。实验失控时，团队被终端视为噪声抹除——美佐是唯一幸存者。她把终端相关的知识编码为 hex tree 藏在了谦治的服务器里——谦治知道它的存在但从没碰过。
- 她选择嫁给谦治——一个和科学界毫无关系的电脑店店主——是她能想到的最安全的生活方式。
- 千夏的书桌最底层抽屉里，有一本美佐的笔记本。千夏从没翻开过——她把它当作「遗物」而不是「可以读的东西」。直到她决定不再只是守着。
- 笔记本不是技术文档——是美佐的私人日记。记录了她嫁给谦治后的日常、对千夏出生的描述、深夜在服务器旁写下的零碎想法。没有密码、没有加密、没有工作内容。只有一个人对家人的爱。
- 千夏翻开日记的时间点：在她已经通过服务器知道母亲是科学家之后。她已经知道了技术真相——但她不知道母亲是一个什么样的人。日记填的不是信息缺口，是情感缺口。

## 姨妈：神楽遥

- 美佐的妹妹。千夏的姨妈。已故（2019 年末）。
- 千夏对遥只有模糊的童年记忆——一个偶尔来店里、和父亲聊很久的阿姨。不知道她是黑客。不知道她和母亲的关系有多深。不知道她死在和母亲相似的境遇里。

## Merge Conflict（新版逻辑）

林奕试图回到过去救谦治 → 谦治活着 → 作为电脑店主，谦治知道服务器上有美佐留下的东西——如果他活着，他会更谨慎地处理 P53 和服务器 → 林奕买不到 P53 → wl0 不触发 → 无法回去救谦治 → **因果死锁**

## 性格

参考质感：《Serial Experiments Lain》玲音。

- 表面平静到近乎空洞。不太会表达，但会记住对方在乎的细节。
- 她的沉默不是「没有想法」——是她在处理。她在脑子里跑的东西比她嘴上说的多得多。
- 和机器的互动比和人更自然。她和服务器的关系不是「守遗物」——是某种她自己也说不清的连接。她在柜台后面看指示灯的时候，和玲音盯着屏幕的时候是同一种表情。
- 在不同的语境中切换不同的自我：在店里是沉默的店主，在服务器房间是一个专注到忘记时间的人，在林奕和叶岚面前会偶尔露出让人停住的洞察。
- 表面被动——但一旦决定了，不会再等。
- 行动比语言快。她不说「我要弄清楚」——她直接开始弄。

## 结局

- 回溯后的新世界线。林奕没有买 P53。千夏不认识他。
- 翔太在柜台后调弦：「认识的人吗。」「……不知道。」千夏低头继续做事。
- P53 在仓库落灰→废品→填埋场→焚烧。A 面刻字在火焰中最后亮了一下。
- 千夏翻一个空本子。合上。继续做事。UPS 绿灯在闪。
- 他们都活着。没有记忆——有残留。
