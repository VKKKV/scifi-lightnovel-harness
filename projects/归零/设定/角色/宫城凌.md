# 宫城凌（Miyagi Ryo）— 配角 2

技术导师 / 独自追查真相的人 / 核心情感引爆点

- 真名：宫城凌（Miyagi Ryo）
- A 面刻字名称：`リョウ`（片假名）
- 年龄：28 岁
- 性别：男
- 身份：前安全研究员，现无业/家里蹲，靠灰黑产获取大额收入
- 运营极其火热的日本 Arch Linux 论坛

## 恋人：神楽 遥（ハルカ）

- 已故（2019 年末）
- Arch Linux 社区核心贡献者，日本榜第一
- 通过浅羽谦治的电脑店定制了一台 P53 顶配——128GB/12TB/RTX5000——作为送给凌的生日礼物
- A 面刻字：`ハルカ と リョウ`
- **凌从始至终不知道这台电脑的存在**——直到六年后，在一个昏暗的 livehouse 里

## 独自追查（新设）

**凌从未持有终端权限。** 遥死后，凌在整理她的遗物时发现了线索——不是终端，是她从姐姐美佐那里继承的观测日志。遥用自己的方式记录了美佐留下的终端相关数据——她看不懂全部，但她持续记录了好几年。凌凭这些推断出了世界上存在一个普通人看不到的系统层级。他换掉了 waterlily 的 ID——不是改名字，是重新变成了她。

他追查了六年。他比林奕更早知道 kiri 的存在。他在林奕之前就撞到了那个 ID 的痕迹，只是他不知道那个痕迹通向什么。他至今没有走到终点。

## 与林奕的相识

**不是传统的「发帖求助 → 大佬回复」**：

1. **触发**：林奕在 Arch Linux 论坛发了一篇技术帖——描述 P53 上新出现的一张虚拟网卡 `wl0` 的路由行为极其诡异。附上了终端报错日志和抓包截图——数据包的时间戳有明显的不符合当前时间的偏移
2. **kagure 看到帖子**：屏幕那头的凌看到 `wl0` 和那串特征流量——他认得。六年前他也见过一模一样的模式。**他立刻删除了帖子**——不是在帮林奕——是在防止更多人看到 `wl0` 这个关键词
3. **第一次私聊**：Kagure 直接私信林奕。没有问候，没有自我介绍——只有三行警告：「删了你的帖。别再发。别再碰那个接口。如果你不听——不要让我知道。」
4. **林奕当然没有听**。黑客的好奇心不会因为三行警告就消失
5. **关系建立**：随后林奕在 CTF 和终端探索中展现出的直觉——让凌看到了曾经的自己。他开始在私信里给林奕回答技术问题——冷冷的，但每次都对。他教林奕的不是 git 命令——是**渗透技术和网络攻防**。git 只是他用来解释「世界线分支」的比喻面板。真正的教学——是如何在目标网络上拿到 shell
6. **此时的凌对林奕用的什么电脑一无所知**。他只知道林奕是「一个用着笨重机器的蠢蛋新人」——从帖子里看到 P53 的 `lsblk` 输出时，他没有想起那台机器。因为他不知道那台机器的存在

## 终极引爆点：延时的命运交汇

- 发生在第一卷末尾或第二卷高潮——林奕和凌因为某个致命危机被迫线下见面
- 场景：秋叶原某个昏暗的地下网咖/安全屋，或是高円寺 livehouse「HIGH」
- 林奕从包里砸出 P53——厚重如砖
- 凌叼着烟，原本还在嘲讽：「都什么年代了，你还背着这种三公斤重的电子垃圾满街跑」
- 然后凌的目光落在了 A 面右下角。灯光昏暗。但银色刻字反了一瞬间的光：`ハルカ と リョウ`
- 他的手停住了。不是抖——是停。烟灰掉在桌面上。他没说话。他把烟按灭
- 这个永远冷静、技术天下第一的黑客——六年来拼命想要忘记的过去，他以为早就被埋掉的东西——以硬件的形式，跨了六年的时间，送到了他面前
- **他才知道——她在那年给他做了一台电脑。他从来不知道。她没来得及说。机器没来得及送。但刻字在了。ハルカ と リョウ。她写下来了。**
