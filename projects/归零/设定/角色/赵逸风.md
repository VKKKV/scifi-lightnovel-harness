# 赵逸风 — 全书唯一「正常人」

## 定位

日常载体 / 正常人视角 / 林奕最「正常」的关系 / 隐藏伏笔库

## 基本信息

- 姓名：赵逸风
- 年龄：20 岁
- 性别：男
- 身份：中国人，和林奕语校同班，合租室友
- 家境：富二代，来日本纯粹是为了玩

## Want / Need / Flaw

- **Want（他想要的）**：想让林奕别老盯着屏幕。不是讨厌计算机——是觉得林奕在把自己活成一段进程。想把他拖出去吃拉面。
- **Need（他不知道的）**：他自己也需要林奕。赵逸风是那种「看起来朋友很多但真正的朋友没几个」的人。他把林奕当成了为数不多的「可以凌晨三点说话的人」——只是他从不说这话。
- **Flaw（缺陷）**：用「大大咧咧」当盾牌。怕认真。怕被当成认真的人——所以永远在开玩笑。但他说的那些让林奕停住的话——不是在开玩笑。

## 四卷弧线

| 卷 | 位置 | 功能 |
|----|------|------|
| 一 | 日常载体 + 林奕的东京锚点 | 奶龙贝利亚鬼图吓得打翻可乐→赔电脑→买P53。凌晨泡面、管人直播。建立：他是林奕最「正常」的关系 |
| 二 | 第一个察觉林奕在变的人 | 不是发现终端——是发现林奕睡得越来越少、话越来越短、眼神越来越远。试过问——被挡回去了。不追问。但在看 |
| 三 | 「意外」事件 → 最终知情 | 赵逸风差点被高空坠物砸中。林奕不能再瞒他。全部说了。他听完——「所以你现在这么累——是因为这个。」他不懂计算机。他懂了人 |
| 四 | 重置后的陌生人——但直觉不对 | 新世界线里他不是室友。但第一次见到林奕时停了一下——不知道为什么。直觉——不是二次元直觉，是对人 |

## 能力

- **博闻强记用错了地方**：几乎看过所有动漫和漫画——不是看过，是熟记。问他《EVA》的世界设定能背出 SEELE 全部成员的名字，问他会写 Python 会发呆。
- **二次元类比引擎**：在日常对话中无意识引用动漫情节——当时只是玩笑，将来会变成现实。后期 callback：当林奕面对某个抉择时，赵逸风某次吃泡面时聊过的那部漫画的结局突然浮现在林奕脑子里。
- **神楽伏笔**：后期某天，老赵无意中看到 P53 的 A 面刻字——「ハルカ？这个发音怎么感觉在哪听过的——」他不是计算机圈的人，但他的二次元脑对名字的直觉——可能会比林奕先踩到那个点上。

## 管人玩家

- 桌面像监控室——多块显示器，多窗口同时看不同 Vtuber 的直播。不是为了实用——是为了同时推好几个箱。
- 后期对应：当你需要同时监控多个世界线的状态，谁的多屏设置最合适？赵逸风的——只是他当时接的不是世界线信号，是管人直播。

## 性格

- 纯二次元，房间里堆满手办和等身抱枕
- 不懂计算机——对林奕的 Arch Linux 的评价是「你这个黑窗口还没装 GUI 吗」
- 乐天派，看起来社交牛逼症——但真正的朋友没几个
- 口头禅是「这不就是那个嘛——」（然后在动画里准确引用对应情节）
- 凌晨三点两人泡面碰到一起——林奕盯着终端，他盯着平板，画风完全无关但莫名其妙和谐
- 用「大大咧咧」当盾牌。偶尔说一句让林奕停住的话——然后马上用笑话盖过去
- 从不深究林奕在干什么。但会在关键时刻问出「你最近是不是不太对劲」——正常人的直觉不知道细节，但能感觉到温度变化

## 叙事功能

- 日常线主要载体 + 节奏调节器
- 提供「正常人反应」作为对照——全书唯一不解析系统的人，他解读人
- 隐藏伏笔库：随口说的动漫桥段将来 callback
- 后期多屏显示器 callback
- 代表「什么都不知道却一直在的朋友」
