# kiri

git log 中出现的神秘 commit 作者。不是林奕。不是 kagure。分布在整个世界线树上，时间戳比林奕任何操作都早——有些早到 2019 年之前。

commit 极简：4 位 hash，message 是数字（1、2、3...）。没有 reset 记录。

kiri 不是一个人——是全球顶尖科学家团队的共同操作账号。ID 含义：日语「切り」（切/斩/结束）。团队建造了超级量子计算机——一台拥有超级 AI、能推演过去未来、干涉因果律的机器。kiri 是他们操作这台机器时使用的共同 ID。

实验失控。一群良心发现的科学家（美佐是其中之一）人为制造了一场爆炸，试图摧毁量子计算机和终端。团队中大部分人在这场爆炸中丧生或被系统视为「噪声」抹除。美佐是唯一幸存者。

爆炸后 kiri 账号不再活动。但 commit 仍然存在于世界线树上——像是量子计算机在爆炸前保存的快照。

commit 特征：
- message 是乱码或极简数字——像是系统自动生成的
- 遍布世界线树——集中在团队活跃的时期
- 从来没有 reset 记录——不需要 reset

与各角色的关系：
- 浅羽美佐：团队成员。良心发现→参与爆炸→唯一幸存者
- 浅羽谦治：不是 kiri。在整理美佐遗物时发现过 kiri 的痕迹
- 浅羽千夏：完全不知道母亲的真实身份
- 林奕：首次在 git log 中看到 kiri 后意识到——这个终端不是他一个人的
- 宫城凌：在追查遥的死因时，撞到了 kiri 的痕迹
- 高柳诚：量子信息科学家，从理论角度推导终端机制
