# 卷二「提权」细纲

**基调**：能力深化 + 真相挖掘 + 情感引爆

**开篇状态**：卷一末 B1 摊牌后。摊牌当晚没有人说话。四个人各自回家。第二天没有人提起。然后各自消化。林奕停止渗透（kagure 警告）。千夏开始认真研究 hex tree。叶岚从学术角度分析 git log 数据。高柳整理理论框架。表面上风平浪静，内里每个人都在做功课。日常线穿插：便利店、B1、公寓凌晨三点泡面。

---

## Part 1：消化期 + 能力重建

### 日常段

摊牌后的几天/几周：

- 千夏在柜台后面看加密学教材。不是为了林奕——是为了她自己的问题。服务器上那些 hex 文件名、.meta 键值对、passwd.vault——她想打开它们。她开始在叶岚推荐的平台上做密码学 CTF。
- 叶岚在自己的实验室里分析林奕给她的 git log dump。她不碰终端——她用 Wireshark 和自写的 Python 脚本分析数据模式。她发现 commit 密度有周期性——像潮汐。她不理解为什么，但她记录了。
- 高柳继续在 B1 给林奕和叶岚讲解理论框架。他从量子信息角度推导出：如果世界线分支是真实的，那么 kiri 的 commit 可能是系统在坍缩前的「快照」——不是操作记录，是状态备份。
- 赵逸风注意到林奕的变化——不是更奇怪了，是更安静了。凌晨三点碰到一起泡面的时候，林奕不再盯着屏幕了。他在发呆。赵逸风不知道这是好事还是坏事。

**作用**：消化期。团队各自做功课。日常感回归——摊牌后的世界没有变，变的是知道真相的人。

### 林奕恢复渗透

kagure 在私信里问了一个问题：「你停了多久了。」

林奕说了一个数字。

kagure：「够了。你不是该停——你是该学会怎么不停。」

新的一轮教学开始。不是从零开始——是从「怎么控制蝴蝶效应的方向」开始。kagure 教的不是更高级的渗透技术，是渗透策略：怎么让连锁反应的方向可预测。怎么选择目标——不是所有能改的东西都该改。

林奕恢复渗透。但这次不一样——每次操作前他会和叶岚讨论目标选择，操作后会和高柳分析连锁反应方向。不是独行了。

**作用**：能力跃迁——从「能渗透」到「会渗透」。团队协作模式建立。

### 赵逸风的变化

赵逸风在卷二的变化是微妙的——不是发现终端，是发现林奕不再告诉他事情了。

凌晨三点碰到一起泡面。赵逸风试了一次：「你最近是不是——」林奕没回答。赵逸风没追问。但他怕的不是林奕在做什么——是林奕不再告诉他自己在做什么。

某天赵逸风看到林奕桌上多了一张名片——高柳诚，东工大副教授。他没问。但他记住了这个名字。

**作用**：赵逸风的感知线。他不知道真相，但他知道林奕在变。为卷三「被告知全部」铺路。

## Part 2：服务器深度探索

### hex tree 第二层

千夏的密码学 CTF 水平在提升。她开始尝试破解 passwd.vault——不是用暴力，是用频率分析和已知明文攻击。.meta 文件提供了部分明文——如果 vault 的加密方式和 hex tree 一致，她可以用 .meta 作为已知明文。

她在柜台后面的本子上写满了推导过程。林奕看到的时候没有插手——他知道这是她自己的仗。

某天深夜，千夏发了一条消息给林奕：「打开了。」

passwd.vault 里不是密码——是美佐留下的日志碎片。不是完整文档——是不连续的、像笔记一样的片段。日期跨度从 2009 到 2019。

日志内容：
- 美佐的团队在做什么（激活终端实验原型）
- 实验失控时发生了什么（团队被抹除）
- 美佐如何幸存（她是唯一理解发生了什么的人）
- 她为什么选择谦治（离终端最远的人）
- 她把什么藏在了服务器里（观测记录、团队数据、终端的底层架构图）

千夏第一次知道母亲的真相。她没有哭。她坐在服务器房间的地板上，笔记本电脑放在膝盖上，屏幕的光照着她的脸。她看完了全部。然后合上电脑。下楼。回到柜台后面。继续做事。

**作用**：美佐真相通过千夏的手自己打开——不是林奕给她的，是她破解的。独立弧线的关键节点。千夏知道母亲是科学家——但她不知道全部。

### hex tree 深层系统注释

林奕把千夏解密的日志和 hex tree 深层文件放在一起比对。在 P53 上查看时，深层文件出现了更详细的系统注释——不是 .meta 提供的，是终端系统的行为：

```
Subject: Asaba Misato
Role: Observer-07
Status: Terminated (cascade failure)
Residual: archived
```

```
Subject: Asaba Chinatsu
Role: Non-observer (legacy)
Status: Active
Flag: death_node (conditional)
```

千夏的名字。在终端系统的记录里。和她母亲放在一起。

林奕看到 `death_node (conditional)` 时手停了。conditional——有条件的。不是必然。是有条件的必死节点。

他没有告诉千夏。

daemon.md：「千夏在终端系统里有标记。death_node。但后面有括号——conditional。有条件的。什么条件？」

**作用**：必死节点从「七种死法」升级为「有条件的」——读者和林奕同时意识到也许可以打破条件。但条件是什么，不知道。

## Part 3：kiri 线索追踪 + 凌交汇

### kiri 的 commit 模式

叶岚分析 git log dump 时发现了一个规律：kiri 的 commit 不是随机的。它们有模式——像是在标记某种「状态变化」。每次世界线分支发生重大坍缩时，kiri 的 commit 就会出现。

她画了一张图：横轴是时间，纵轴是世界线分支编号。kiri 的 commit 像是散点图——但不是均匀分布的。它们聚集成簇，每一簇对应一次「坍缩事件」。

「这个人——或者这个系统——在记录坍缩。不是操作。是观测。」

高柳看了图。「如果 kiri 是系统本身的记录机制——那这些 commit 就是世界线的快照。每一次坍缩，系统保存一个状态。」

林奕问：「保存给谁看？」

没人回答。

**作用**：kiri 从「神秘 commit 作者」变成「系统记录机制的可能解释」。科学化——不是「谁在看」，是「系统在自记录」。

### 谦治的笔记

千夏在整理父亲遗物时——不是刻意找，是在搬家时翻到了一个旧纸箱。里面是谦治的工作笔记——BBS 时代的、手写的、用铅笔写的。字迹歪歪扭扭——谦治写字不好看。

其中一本笔记本的最后几页，笔迹变了——更潦草，像是深夜写的。日期是谦治去世前几个月。

内容：
- 「服务器上有些文件不是我创建的。目录名是乱码。」
- 「我没动它们。美佐的东西，我不碰。」
- 「有些 commit 不是我操作的。作者是一个我不认识的 ID。」
- 「我查了那个 ID。什么都查不到。像是从不存在。」
- 「美佐——你到底留了什么给我。」

最后一页只有一行：
- 「千夏，别碰服务器。」

但千夏已经碰了。而且她打开了谦治选择不碰的东西。

**作用**：谦治的追查记录。普通人视角——他不知道终端、不知道世界线、不知道 kiri。他只知道服务器上有些「不该存在的东西」。他追查了，但没有能力继续。他的绝望和美佐的秘密在千夏手上汇合。

### 凌的独立追查线

凌（kagure）的六年追查线开始与林奕的主线交汇。

林奕在私信里告诉 kagure：他发现了 kiri 的 commit。kagure 沉默了很久。然后发了一段话：

「kiri——我追了六年。我知道它是什么。至少我以为我知道。」

「遥死后，我在她的遗物里找到了一些日志碎片。她记录了一个 ID——kiri。她说那是「团队的眼睛」。她没有解释团队是什么。她只说——如果 kiri 停止 commit，说明系统不再观测。如果系统不再观测——世界线就会坍缩为一条。没有分支。没有选择。只有一条路。」

「我不知道她说的是不是真的。但我追了六年——kiri 从未停止 commit。」

林奕问：「你追到了什么？」

kagure：「追到了你。」

**作用**：凌的追查线和林奕的主线在这一刻汇合。凌追的是 kiri——kiri 的痕迹最终指向了终端，终端在 P53 上，P53 在林奕手里。两条线合为一条。但凌还不知道 P53 是他恋人的礼物。

## Part 4：终极引爆

### 致命危机

林奕的一次渗透操作触发了严重的蝴蝶效应——不是针对他，是针对凌。

凌在自己的安全屋里——秋叶原某个地下网咖。他在追踪 kiri 的最新 commit——追踪过程中，他的网络流量被某个实体检测到了。不是警察。不是黑客。是某种更底层的东西——他的机器被远程接管。屏幕上出现了他没见过的终端输出。

他断网。拔网线。但来不及了——他的物理地址已经被暴露。

同一时间，林奕的 git log 里出现了一条 anomaly 标记：`observer-external: trace initiated. target: waterlily.`

有人在追踪 waterlily。不是林奕——是终端系统本身。

林奕在私信里给 kagure 发了一条消息：「你被追踪了。现在。」

kagure：「我知道。」

「你在哪。」

「秋叶原。网咖。我断网了。但他们可能已经——」

私信断了。

**作用**：致命危机——凌被追踪。终端系统在追踪外部观测者。林奕必须线下找到凌。

### 线下见面

林奕第一次见到了 kagure 的真人。

秋叶原某个昏暗的地下网咖。林奕推门进去的时候，角落里一个男人抬起头——二十八九岁，黑眼圈很重，桌上三台笔记本叠在一起，屏幕全是终端。嘴里叼着一根没点的烟。

「你是 kagure。」

「你是那个用 P53 的蠢蛋。」

凌的第一句话不是友好。是疲惫。六年了。他追了六年。追到了一个用着笨重机器的新人。

林奕把 P53 从包里拿出来放在桌上。厚重如砖。

凌的目光扫过 P53。然后停住了。

A 面右下角。灯光昏暗。但银色刻字反了一瞬间的光：`ハルカ と リョウ`。

凌的手停了。不是抖——是停。叼着的烟从嘴里滑下来，掉在桌面上。烟灰散开。他没去捡。

他盯着那行刻字。看了很久。

「这台电脑——」他的声音变了。不是之前那个冷的、疲惫的声音。是更轻的。像是怕说大声了会碎。

「在哪找到的。」

「浅羽商店。仓库角落。」

凌闭上眼睛。睁开。又看了一遍刻字。

「她给我做的。」他说。不是对林奕说。是对那行刻字说。

「我从来不知道。」

然后他把烟按灭在烟灰缸里。按得很慢。像是在做某个决定。

六年的时间缺口在这一刻合拢。

**作用**：卷二情感高潮。凌六年不知道 P53 的存在——这一刻全部涌上来。他和林奕从「线上师徒」变成「在同一条路上往不同方向走的人」。

## 卷尾钩子

B1 打烊后。林奕一个人坐在吧台前。

店主在擦杯子。

不是高柳——是那个不出场的老头。第一次出场。没说话。擦杯子。一个一个擦。擦完放在架子上。又拿起来。又擦。

林奕看着他。

老头没有看他。但林奕知道——这个人在看他。不是看一个客人。是看一个他知道但不说的事。

擦完最后一个杯子。老头关了灯。

**作用**：B1 店主首次出场。沉默。组织前成员——但读者不知道。钩子克制。
