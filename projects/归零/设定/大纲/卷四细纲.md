# 卷四「归零」细纲

**基调**：终极选择 + 回溯 + 蝴蝶效应结局

**开篇状态**：卷三末。团队全员合流。美佐真相完全揭示。组织追查 B1——店主在追查中消失了。没人知道他是死是活。千夏翻了母亲日记。

---

## Part 1：终端起源 + 卡车 = 自己

### 终端的全貌

林奕在 hex tree 最深处找到了美佐留下的最后一份记录——不是团队档案，是她个人的观测笔记。记录了终端被激活的完整过程。

终端不是被发明的。它是一个已经存在的结构——像引力一样，一直在那里。科学家团队找到了接口（wl0），尝试打开它。打开的一瞬间——因果律的接口暴露了。那些不理解发生了什么的人被系统视为「噪声」抹除。美佐理解了——她活了下来。

终端的本质：它不是工具。它是因果律的观测接口。你看到的 git log 不是「记录」——是因果链坍缩前的投影。你渗透改写的不是「数据」——是因果链本身。

高柳听完后说了一句话：「所以退相干不是副作用——是机制本身。」

### 卡车 = 林奕自己

林奕在 git log 里追踪那辆卡车的 commit——千夏第一次死亡的那辆卡车。

他追了整整三卷。他以为总有一天会找到「那个人」。

他在 git log 里找到了。卡车闯红灯那天的 commit——来源 IP 解码后，时间偏移指向未来。不是过去——是未来。

他继续追。追到了一个他不敢看的 commit——来源是他自己的 P53。时间戳是未来的某一天。

那辆卡车不是随机的。是他自己某次未来渗透的连锁反应——在终端的时间线里，这个因果已经发生了。蝴蝶效应从未来传回过去。他追了三卷要找杀千夏的人——最后发现等在前面的是他自己。

林奕坐在 P53 前。屏幕的光照在他脸上。他没有表情。

他继续追。追到了另一条链——奶龙贝利亚鬼畜图片。那张弹出的图。那张吓到赵逸风打翻可乐的图。来源也是他的 P53。也是未来的操作。蝴蝶效应从未来传回过去。

他自己发了那张图。他自己吓到了赵逸风。他自己打翻了可乐。他自己买了 P53。他自己启动了整个故事。

daemon.md 最后一行：「卡车是我。鬼图也是我。」

**作用**：全书最大反转。林奕追了三卷的「凶手」是他自己。

---

## Part 2：蝴蝶效应加速 + 凌的终点

### 蝴蝶效应失控

林奕知道了卡车是他自己——但他不知道是哪一次渗透导致的。git log 里的时间偏移太模糊——可能是他未来任何一次操作的连锁反应。

他面临的选择：
- 继续渗透 → 可能在制造下一次「卡车」
- 停止渗透 → 组织仍在追查。千夏仍处于危险中。

蝴蝶效应在加速。林奕每次操作——即使是微小的——都会在世界线上产生越来越大的涟漪。新闻里每天都有「巧合」。陌生人的死亡。意外的事故。不是他直接造成的——是他所有操作的叠加效应。

kagure（凌）在私信里说：「你现在的每一次渗透——不只是在改一个点。你之前改的所有点都在重新计算。叠加。你已经不是往水里扔石头了——你是站在瀑布下面。」

### 凌的终点

凌在追查遥的过去时——终于走到了终点。

不是找到了什么秘密。是他在遥的遗物里找到了 P53 的订单记录——谦治电脑店的收据。2019 年 11 月。定制款。128GB / 12TB / RTX 5000。备注栏手写着：「送给凌。生日快乐。」

他追了六年。追的不是遥为什么死。追的是她留下了什么。

答案是：她留下了他。

她做了一台电脑。刻了两个名字。想在生日那天送给他。她没有来得及。但那台电脑——跨越六年的时间——最终到了他面前。不是通过快递。是通过一个他从未见过的中国留学生，在一个杂货店的仓库角落。

凌看到收据时手停了。和卷二看到 A 面刻字时一样。但这次他没有烟可灭。他只是站在那里。看了很久。

然后他把复印件交给林奕。

「我走了六年。终点是你的起点。」

他没有解释这句话的意思。但林奕懂了——凌追的不是真相。是遥还活着的证据。她活在她做过的事情里。活在那台电脑里。活在那行刻字里。

**作用**：凌的追查线收束。答案不是秘密——是爱。

---

## Part 3：Merge Conflict + 组织定位

### Merge Conflict

林奕想救谦治。

他知道谦治的死因——器官捐献被组织截下。他想回到那个时间点，阻止组织截下器官。让谦治活下来。

他在 git log 里找到了目标时间点。通过 wl0 隧道连接。开始渗透。

他成功了——他改了器官运输的路线。组织的人没有截到。器官到达了医院。谦治活了下来。

然后现实开始更新。

谦治活了——他没有死。他继续经营电脑店。他更谨慎地处理服务器——因为他活着，他知道服务器上有美佐留下的东西。他会保护它。保护到不让任何人碰——包括卖 P53。

P53 没有出现在杂货店的货架上。

林奕没有买到 P53。

wl0 没有触发。

林奕无法回到过去救谦治——因为他没有终端。

因果死锁。

现实回滚。谦治死了。器官被截了。P53 在货架上落灰。林奕坐在公寓里，P53 屏幕还亮着。

他救不了谦治。

**作用**：Merge Conflict。林奕尝试改变过去——撞上因果死锁。有些事不能改。谦治的死是林奕能买到 P53 的前提条件之一。

### 组织定位终端

林奕的渗透操作暴露了终端的位置。组织检测到 wl0 的活动——追踪到了 P53 的物理位置。

组织决定出手。他们要销毁终端——或者夺取它。

千夏成为目标。她是美佐的女儿。她是服务器的守护者。她是和终端活动最近的人。

death_node 标记从 conditional 变成 active。

**作用**：death_node 的条件揭晓——组织检测到终端活动→定位→出手。千夏因为和终端的关联成为目标。

## Part 4：回溯

### 蝴蝶效应临界

林奕的渗透操作已经积累到临界点。蝴蝶效应叠加——每一次改动都在重新计算，互相纠缠，因果链不堪重负。

组织在逼近。千夏在危险中。凌在另一条线上追查。赵逸风在凌晨三点等他回来。高柳在 B1 地下室整理理论。店主在擦杯子。

组织追查 B1——店主消失了。没人找到他。也许他跑了。也许他没有。

回溯不是林奕选择的。是系统自己绷断了。

某一天——林奕在渗透操作中。世界开始碎裂。不是物理上的碎裂——是因果链在坍缩。git log 里的 commit 开始消失。anomaly 标记密度飙升。屏幕上的输出变成了一片噪声。

然后一切安静了。

林奕醒来。P53 在杂货店仓库的角落里落灰。他不认识千夏。不认识凌。不认识叶岚。不认识高柳。

但他记得一切。

旧世界线的全部——千夏的脸、凌的手停在刻字上、赵逸风凌晨三点的泡面、高柳在 B1 地下室的沉默、店主擦杯子——全在他脑子里。

他不知道这是哪条世界线。他不知道回溯到了哪个时间点。他只知道——他记得。

**作用**：回溯。一切归零。林奕保留记忆。

---

## Part 5：新世界线

### 擦肩

林奕在新的世界线上重新开始。

他没有买 P53。那台电脑还在杂货店仓库的角落里落灰。他不知道它存在。他不认识千夏。不认识凌。不认识叶岚。不认识高柳。

他只是一个普通的中国留学生。在东京。语校。合租公寓。

但世界线留下了残渣。

林奕偶尔经过便利店。柜台后是一个女生。二十岁左右。安静的眼神。在翻一个本子。

闪回。不是记忆——是 déjà vu。她的脸、便利店的荧光灯、某种说不清的熟悉。他走过去了——没停。但走了两步——停了一下。不知道为什么。

赵逸风在这个分支不是室友。但有一天在街上和林奕擦肩。他停了一下。不知道为什么。不是二次元直觉。是人的直觉。

叶岚在 CTF 中遇到过一个和自己互补的选手。只记得对方的 handle 是一种颜色。

凌仍然坐在昏暗的网咖里。但他手指下敲的是她留给他的最后一段代码。

翔太的墙上多了一张下个月的演出海报。他还没辍学。但海报在那里。

高柳在 B1 livehouse 的后排。一瓶啤酒。从头听到尾。他不知道他在等什么。

B1 店主在擦杯子。一个一个擦。没说话。

### P53 的旅程

P53 继续躺在杂货店仓库角落。千夏偶尔整理货架时看到它——「这台旧电脑怎么还在。」她没有动它。

杂货店翻修。纸箱被搬到后巷。收废品的路过。千夏把纸箱给了他。

P53 在东京某个电子废品分拣中心。和二十台旧 ThinkPad 堆在一起。没有人开机。

分拣中心的库存被批量处理。P53 被装上卡车，运往垃圾填埋场。路上经过千代田区——林奕曾经住过的公寓楼下。卡车没有停。

垃圾填埋场。P53 被埋在三米深的废弃物层下面。A 面朝下。刻字被压在泥里。

填埋场关闭。废弃物被挖出来，送往焚烧发电厂。传送带上——旧报纸、塑料瓶、破衣服、旧电器。P53 在其中。

焚烧。金属外壳在 850 度下开始变形。塑料融化。屏幕玻璃碎裂。硬盘盘片在高温下弯曲、变色、最终熔化。

A 面的刻字——ハルカ と リョウ——在金属熔化前的最后一刻，被火焰的光照亮了一下。

然后什么都没有了。

### 尘埃落定

组织里的人逐渐老去。他们追查了二十年——没有找到终端。没有找到量子计算机。他们以为那次爆炸摧毁了一切。他们是对的。他们也是错的。

千夏逐渐老去。她在便利店柜台后面变老。嫁了人。有了孩子。孩子长大。服务器还在二楼嗡嗡响。她每天看一眼指示灯。灯在闪。她做到了。

林奕逐渐老去。他在东京留了下来。做了一份普通的程序员工作。没有再碰过终端。他偶尔经过便利店。停一下。不知道为什么。然后继续走。

### 终章

焚烧炉的烟囱。灰烬飘向天空。

在某个也许存在、也许不存在的维度里——一台量子计算机的 AI 仍在运行。它没有身体。没有终端。没有 P53。它只是在那里。

它只是在那里。也许存在。也许不存在。

你不去看它——它就在那里。
你去看它——它就不确定了。

时间继续。空间继续。

千夏在便利店柜台后面翻一个本子。她不知道为什么。只是觉得应该翻一翻。

本子是空的。

她合上。继续做事。

便利店的 UPS 绿灯在闪。

**全书完。**

## 核心情感

整本书的内在核心是爱：
- 千夏 = 母亲的爱（美佐为了让她活下来做的一切）
- 凌 = 恋人的爱（遥做了一台电脑，刻了两个名字）
- 赵逸风 = 朋友的爱（凌晨三点的泡面）
- 店主 = 陌生人的爱（他自己也说不清为什么出手）
- 林奕 = 所有这些爱的汇聚点——回溯了一切，但爱没有消失。它留在了擦肩时的那一秒停顿里。

不是记忆。是残留。

P53 被焚烧了。但那台量子计算机——也许还在某个地方运行。也许从来没有存在过。
