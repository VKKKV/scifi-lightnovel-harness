# 卷三「越权」细纲

**基调**：真相揭露 + 冲突爆发 + 全员合流

**开篇状态**：卷二末。凌和林奕线下见面。A面刻字揭示——凌知道 P53 是遥做的。凌从「线上师徒」变成「同一条路上的人」。B1 店主首次出场（擦杯子）。千夏破解了 passwd.vault，知道母亲是科学家。

---

## Part 1：美佐记录 + 科学家团队

### hex tree 深层解码

卷二解开了 passwd.vault（美佐日志碎片）。卷三继续深入 hex tree。

叶岚和凌合作——叶岚用流量分析技术追踪 hex tree 文件之间的数据关联，凌用渗透经验尝试突破加密层。两个人的方法不同但互补：叶岚从外部观测，凌从内部渗透。

更深的文件开始揭示科学家团队的全貌：
- 团队由十二人组成——来自量子物理、计算机科学、密码学、神经科学
- 他们不是发明了终端——是激活了一个已经存在的结构。终端像是某种「基础设施」——在人类出现之前就存在。团队找到了接口，尝试打开它。
- 激活的一瞬间，团队中大部分成员被系统视为「噪声」抹除。只有少数人幸存——美佐是其中之一。

林奕在 P53 上看到系统注释：
```
Project: OPEN_GATE
Status: FAILED (cascade)
Survivors: 1/12
Observer-07 (Misato): active
```

高柳看到这条记录时停了很久。只有一个幸存者。美佐。

**作用**：科学家团队全貌揭示。终端不是被发明的——是被激活的基础设施。美佐是唯一的幸存者。

---

### 凌的追查线合流

凌在团队中逐渐找到自己的位置。他和高柳有化学反应——两个人都追了六年，只是方向不同。高柳从学术角度推导，凌从遗物碎片追踪。

凌开始在 B1 地下室出现。不是每次都来——但他来的时候，会在角落坐着，不说话，听高柳讲理论。偶尔插一句：「你说的退相干——遥的笔记里提过。她说的不是退相干。她说的是「坍缩前的回声」。」

高柳停下来。「回声？」

凌从口袋里掏出一张纸——遥的笔记碎片。六年前的。纸已经发黄了。上面写着一行字：「kiri commit 不是记录。是回声。坍缩前的回声。」

高柳看了很久。「如果她说的是对的——那 kiri 的 commit 不是系统在记录过去。是系统在预测未来。」

这个发现改变了所有人对 kiri 的理解。

**作用**：凌和团队合流。遥的笔记和高柳的理论碰撞——kiri 从「记录机制」变成「预测机制」。

---

## Part 2：美佐身份全面揭开

### 美佐 = 最后幸存者

林奕在 hex tree 最深层找到了美佐的完整记录——不是日志，是团队的官方档案。

美佐在团队中的角色：她是唯一从项目中期开始质疑方向的人。当其他人兴奋于「打开了因果律的接口」时，美佐是第一个说「我们应该关掉它」的人。

她没有被抹除——因为她理解发生了什么。系统不抹除理解者。它只抹除噪声——那些不理解但碰触了接口的人。

美佐逃了出来。嫁给了谦治——一个和科学界毫无关系的电脑店店主。她把终端相关知识编码进服务器——不是为了保存，是为了「如果有一天有人需要它，它在那里」。

千夏听到这些的时候，坐在 B1 地下室的角落。手放在膝盖上。没有哭。叶岚坐在她旁边——没有说话。只是在那里。

**作用**：美佐从「千夏的母亲」变成一个完整的人——科学家、质疑者、幸存者。

---

## Part 3：组织活跃化 + B1 店主出手

### 意外包围

组织开始行动。他们发现了终端的活动——林奕的渗透操作留下的痕迹。

意外开始：
- 赵逸风差点被高空坠物砸中——有人提前十分钟把垃圾桶挪了位置。坠物砸在他身后三米。
- 叶岚的实验室「意外」断电——备份服务器全部离线。她三天的工作丢失。
- 千夏便利店门口多了一辆停了三天没动的车。车里没人。但每天早上车的位置都不一样。
- 高柳在回家路上发现有人跟踪——不是人，是无人机。小型的，飞得很高。他用手机拍到了。

林奕在 git log 里看到了异常：一条 commit，message 只有一个字：「挡」。不是他的操作。

**作用**：组织的手段——不杀人，改变概率。B1 店主出手的证据。

### B1 店主暴露

林奕追查那条「挡」的 commit——来源指向 B1 livehouse 的物理位置。

他去 B1。不是演出日。店主在。

林奕坐在吧台前。店主擦杯子。没有对话。

林奕问：「你救了赵逸风。」

店主没有停下手上的动作。擦完一个杯子。放在架子上。又拿了一个。

「为什么。」

店主看了他一眼。第一次。眼睛不是老年人的眼睛——是见过太多东西的眼睛。

「你不需要知道。」他说。声音很轻。像是很久没说话了。

然后他继续擦杯子。

林奕没有追问。但他知道——这个人在保护他们。代价是组织发现了他还活着。

**作用**：店主和林奕的第一次正面接触。克制——不解释动机。

### 赵逸风被告知

赵逸风那件事之后，林奕不能再瞒他。

凌晨三点。公寓。泡面。

林奕说了一切。终端。渗透。蝴蝶效应。千夏的 death_node。

赵逸风听完沉默了很久。筷子在碗里搅——不是在吃，是在想。

然后他说了林奕听过的最简单的一句话：「所以你现在这么累——是因为这个。」

他不懂计算机。他懂了人。

「你需要我做什么。」

「什么都不需要。」

「那我继续给你泡面。」

**作用**：赵逸风知情。他的反应不是恐惧/震惊——是确认。他早就感觉到温度变化了。现在他知道原因了。

## Part 4：千夏翻日记

千夏在某个夜晚翻开了母亲的笔记本。

不是因为需要线索——是因为她已经知道了母亲是谁。技术真相从服务器里挖完了。但她不知道母亲是一个什么样的人。

笔记本是日记。美佐嫁给谦治后的日常。

「谦治修板子的时候会哼歌。不好听。但我没告诉他。」

「千夏第一次叫妈妈的时候我正在改一行代码。我把那行代码删了重写了一遍——手在抖。」

「如果有一天千夏看到这本日记——妈妈想让你知道。我做的所有选择，都是为了让你能在一个正常的世界里长大。我没有做到。但你做到了。」

千夏坐在便利店二楼。服务器在旁边嗡嗡响。她一页一页翻。没有哭。但翻到最后一页的时候——手停了。

**卷尾钩子**：千夏合上笔记本。美佐没有留下「密码」——留下了她作为一个人活过的证据。
