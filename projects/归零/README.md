# 归零

科幻悬疑 + 都市日常 + 黑客/计算机文化。四卷 ~400 章 ~160 万字。

## 项目文件

```
归零/
├── constraints.md       ← 项目专属硬约束（R-G 系列规则）
├── README.md            ← 本文件
├── 设定/                ← 世界观/角色/大纲（与 Harness 平级，独立维护）
├── 状态/
│   ├── 主角.md          ← 林奕当前状态
│   └── 角色/            ← 其他角色状态
├── 记忆/
│   ├── 事件索引.md
│   ├── 伏笔登记表.md    ← 伏笔追踪
│   ├── 风格参考.md
│   ├── 日常场景库.md
│   └── 章节摘要/        ← 每章 200 字摘要
├── 大纲/
│   └── 细纲/            ← 逐章细纲
└── 正文/                ← 各章正文（.md）
```

## 创作原则

1. 剧情和人物第一，读者第二
2. 人物 Want/Need/Flaw 先于情节设计
3. 日常场景有功能性——不是灌水，是建立人物关系和节奏
4. 大情感用小动作——用控制写失控
5. 每天写 3000-4000 字一章
