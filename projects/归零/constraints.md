> 所有角色名、伏笔清单、世界观细节均在此维护，不内嵌在 harness 主体文件中。

## R-G 系列：归零专属硬约束

### R-G1 · 终端不拟人（硬）

终端没有任何拟人消息、提示、警告。没有「系统」在引导角色。
林奕知道的所有规则 = 他自己推导的。
违规 = ★★★ 致命。

### R-G2 · 唯一操作 = CTF 渗透（硬）

改变现实的唯一方式 = 通过 wl0 隧道连接过去时间点网络 → 进行黑客渗透。
git 命令（log/reset/branch）只读——观测面板，不改变现实。
违规 = ★★★ 致命。

### R-G3 · 蝴蝶效应代价（硬）

每次渗透 → 历史从修改点重新计算 → 连锁反应超出预测。
渗透规模越大 → 波及范围越广 → 林奕周围的人被牵连。
不是系统惩罚——是因果律没有安全边界。
违规（如「渗透后无事发生」）= ★★★ 致命。

### R-G4 · 闪回 = déjà vu（硬）

被抹除的世界线分支留下闪回碎片——déjà vu，不是完整记忆。
大脑只是辅助信号——大部分操作知识靠 git log 和笔记。
不能写成「林奕想起了另一条线上的完整对话」。
违规 = ★★★ 致命。

### R-G5 · 唯一持有者（硬）

同一时刻只能有一人持有终端权限。目前只有林奕。
历史上的前任已全部失去权限（死亡/quit）。
没有同时存在的「多个观测者」。
违反 → ★★★ 致命。

### R-G6 · 科学自洽层级（硬）

量子解释只从高柳教授（量子信息科学）和 ytbjplh 团队视角给出。
林奕自己的理解停在 git 比喻层（卷一相信 git=真相，卷二开始怀疑）。
不允许林奕自己解释退相干/多世界诠释——那是教授的领域。
违规 = ★★ 严重。

### R-G7 · 日常主线比例（软）

主线（渗透操作+真相探索）：~50%
日常（计算机社区文化+人物互动+教学+感情线）：~50%
每卷偏离不超过 10%。连续 5 章纯日常/纯主线 → 警告。
违规 = ★ 建议。

### R-G8 · 信息释放节奏（硬）

卷一：和林奕一样——通过试错推导规则。不提前解释机制。
卷二：高柳教授+kagure 视角——开始从物理和技术角度理解。
卷三：框架解释——但仍有异常未解。
卷四：终极真相——几十年前实验的建造者和原因。
不允许在对应卷之前提前泄露后续卷的信息层级。
违规 = ★★★ 致命。

## 叙事硬约束

- 第一人称：不允许。第三人称限知视角（以林奕为主，偶尔切千夏/凌）。
- 「系统不说话」：终端不给消息。不叮嘱读者。
- 闪回 ≠ 完整记忆：不允许「他想起了一条完整的世界线」。
- 蝴蝶效应 ≠ 随机厄运：每次连锁反应有可追溯的因果链（可隐约/模糊）。
- 情感用控制写：越大的情感 → 越小的动作。不允许流泪/瘫坐/大段独白。
- 量子术语只在高柳教授解释时出现：设定本身用「藏了知识」「改了分支」「她死了」。
- 唯一一抹奇幻：终端为何选中林奕——不解释。留白。不能用第二次。
