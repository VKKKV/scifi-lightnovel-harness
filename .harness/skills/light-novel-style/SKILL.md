# 轻小说风格审查

> 审查轻小说特有的风格问题：对话自然度、角色魅力、情感表达克制、避免翻译腔。

## 检查维度

### 1. 对话自然度
- □ 是否有翻译腔（「嘛」「的说」「真是的」「……呢」过度使用）？→ ★★ 严重
- □ 角色对话是否千人一面（不同角色用相同的语气/句式）？→ ★★ 严重
- □ 对话是否为了向读者解释而存在（「如你所知……」）？→ ★★ 严重
- □ 技术角色是否在日常对话中滥用术语？→ ★ 建议

### 2. 角色魅力
- □ 每个角色的语言习惯是否一致且可辨识？→ ★ 建议
- □ 角色反应是否符合已建立的性格？（如凌不会失态）→ ★★ 严重
- □ 对话是否有「AI 感」（过度礼貌/过度完整/无省略）？→ ★ 建议

### 3. 情感表达
- □ 是否有尬煽（流泪/大段内心独白/echo 式结局）？→ ★★★ 致命
- □ 大情感是否用大动作（瘫坐/手抖/击倒）而非小动作？→ ★★ 严重
- □ 「用控制写失控」原则是否被遵守？→ ★ 建议

### 4. 叙事节奏
- □ 是否连续多段无对话（纯叙述块 > 500 字）？→ ★ 建议
- □ 每章是否有至少一个「呼吸点」（日常/幽默/温暖瞬间）？→ ★ 建议
- □ 高潮段落是否用短句短段加速？→ ★ 建议

### 5. 避免 AI 式轻小说
- □ 是否有「内心 OS」式括号吐槽过度使用？→ ★ 建议
- □ 是否有角色类型化（傲娇/天然呆/腹黑贴标签而无线索）？→ ★★ 严重

## 违规等级

| 等级 | 标记 | 含义 |
|:-----|:-----|:-----|
| 致命 | ★★★ | 尬煽——踩用户红线 |
| 严重 | ★★ | 风格偏差——需确认 |
| 建议 | ★ | 优化建议——可选 |

## 输出格式

```
━━ 轻小说风格审查 ━━
审查范围：第X-Y章
★★★ 致命 / ★★ 严重 / ★ 建议
  [位置] 问题 → 规则依据
```
