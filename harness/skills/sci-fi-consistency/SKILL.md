# 科学自洽审查

> 审查科幻设定的内部逻辑一致性。确保世界线系统、蝴蝶效应、终端机制的规则在正文中自洽。

## 检查维度

### 1. 终端行为一致性
- □ 终端是否在任何场景下拟人化（发消息/提示/警告）？→ ★★★ 致命
- □ git 命令是否被用于直接修改现实？→ ★★★ 致命
- □ 唯一操作方式是否为 CTF 渗透？→ ★★★ 致命
- □ 同一时刻是否有多于一个持有者？→ ★★★ 致命

### 2. 蝴蝶效应因果链
- □ 每次渗透的连锁反应是否有可追溯的因果链（可模糊，不可无）？→ ★★ 严重
- □ 代价是否波及主角周围的人（不是系统惩罚主角本人）？→ ★ 建议
- □ 蝴蝶效应的规模是否与渗透规模匹配？→ ★ 建议

### 3. 闪回一致性
- □ 闪回是否被写成完整记忆（「他想起了一条完整的世界线」）？→ ★★★ 致命
- □ 闪回内容是否超过 déjà vu 的程度？→ ★★ 严重
- □ 闪回频率是否与渗透次数不成比例？→ ★ 建议

### 4. 量子术语层级
- □ 角色是否解释了自己不该懂的量子概念？→ ★★ 严重
- □ 量子术语是否出现在林奕/千夏的内心独白中？→ ★★ 严重
- □ 教授的理论解释是否与设定文件一致？→ ★★ 严重

### 5. 时间循环逻辑
- □ Bootstrap 循环是否存在矛盾（未来操作和当前状态冲突）？→ ★★ 严重
- □ Merge Conflict 场景（救谦治→死锁）的逻辑是否成立？→ ★★ 严重

## 违规等级

| 等级 | 标记 | 含义 |
|:-----|:-----|:-----|
| 致命 | ★★★ | 违反核心设定——必须修改 |
| 严重 | ★★ | 逻辑矛盾——需要确认或修改 |
| 建议 | ★ | 轻度偏离——可选修改 |

## 输出格式

```
━━ 科学自洽审查 ━━
审查范围：第X-Y章
★★★ 致命 (N 处)
  [位置] 问题描述 → 规则依据
★★ 严重 (N 处)
  [位置] 问题描述 → 规则依据
★ 建议 (N 处)
  [位置] 问题描述 → 规则依据
```
